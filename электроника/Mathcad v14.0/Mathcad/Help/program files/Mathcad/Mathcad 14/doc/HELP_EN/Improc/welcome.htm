The Help window for this Extension Pack has three tabs for accessing information:

  • The Contents tab gives you access to a Table of Contents which is useful when you need an overview of topics available and a guide through them.

  • The Index tab directs you to specific topics you request. Use the Index if you know the topic you are looking for.

  • The Search tab searches the entire help system for a word or phrase you request. Full-text searching is useful when you are looking for a certain word or phrase, rather than a topic.

When you double-click on a topic, it displays in the right-hand side of the window.
