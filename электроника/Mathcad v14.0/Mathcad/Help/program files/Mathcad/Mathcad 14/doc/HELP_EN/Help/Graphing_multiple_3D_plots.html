Graphing Multiple 3D Plots

To plot more than one 3D plot in the same graph:

  1. Define the arguments (matrices or vectors) needed for the plots you want to create.

  2. Choose Graph from the Insert menu, and pick the kind of 3D plot you want to create.

  3. In the placeholder of the 3D plot operator, enter the arguments for the first 3D plot.

  4. Type a comma.

  5. Enter the arguments for the next 3D plot.

  6. Continue entering arguments separated by commas. If any of the 3D plots requires more than one argument, enclose those arguments in parentheses.

  7. Press [Enter] or click outside the plot region.

  8. Double-click on the graph to open the 3D Plot Format dialog box. Use the Display As section and the Plot 1, Plot 2, and other tabs to specify how each set of arguments should be displayed.
