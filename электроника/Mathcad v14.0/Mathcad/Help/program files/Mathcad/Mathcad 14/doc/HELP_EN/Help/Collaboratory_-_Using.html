The Mathcad User Forums

PTC provides Users Forums where you can communicate with other Mathcad users with similar interests and issues. You can post questions, upload or download Mathcad and other files, and connect with Mathcad users around the world. The User Forums allow you to:

  • participate in threaded discussions in a variety of disciplines and topics,

  • post messages and files,

  • read messages and files contributed by other Mathcad users.

Logging into the Mathcad User Forums

Open the User Forums by choosing User Forums from the Help menu or going directly to http://www.ptc.com/go/mathsoft/collab/.

To log in as a new user:

  1. Click the "New User" button.

  2. Fill out the New User form. PTC does not use this information except as necessary to allow you to use the Mathcad User Forums.

  3. Wait for an email message with your login name and password.

  4. You can now return to the User Forums login screen and log in as an existing user.

Notes:

  • Under More on the User Forums menu bar, you can edit your profile to change your password.

Additional Features

To create an address book, mark messages as read, view certain messages, and request automatic email announcements when specific forums have new messages, choose More from the User Forums menu bar.

For more information on these and other features, click Help on the User Forums menu bar.

Editing Your Profile

To change any of your information or make changes to the User Forum defaults, you need to edit your profile.

You can change information such as your login name and password. For privacy when posting messages, you can hide your email address.
