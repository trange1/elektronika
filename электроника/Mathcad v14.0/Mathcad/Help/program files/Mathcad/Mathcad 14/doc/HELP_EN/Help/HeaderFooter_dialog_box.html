Header and Footer Dialog Box

Both headers and footers contain the following options:

Left, Center, Right text boxes

Type in the text or symbols that you want to appear. Click "Use full width for each section" if you want to specify long lines of text in the left, right, or center boxes.

Tools

Inserts the name of your worksheet. Displays as {f}.

Inserts the path to your worksheet. Displays as {p}.

Inserts the page number. Displays as {n}.

Inserts the total number of pages in your worksheet. Displays as {nn}.

Inserts the date when your worksheet was last saved. Displays as {fd}.

Inserts the time when your worksheet was last saved. Displays as {ft}.

Inserts the current date. Displays as {d}.

Inserts the current time. Displays as {t}.

Format

Specifies the font.

Image

Inserts a bitmap file. Displays as {image}.

Start at page number

Specifies the page number for the first page.

Different header and footer on first page

Displays two additional tabs used to format a different header or footer for the first page of the worksheet.

Page

Prints a border around each page.

Header

Prints a border around the header.

Footer

Prints a border around the footer.
