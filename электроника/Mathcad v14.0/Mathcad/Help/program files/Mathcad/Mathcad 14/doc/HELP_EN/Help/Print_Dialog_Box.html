Printing

Print on the File menu lets you print the entire worksheet, a range of pages, only the regions you have selected, or the currently active page. The settings you choose affect only the current print job.

Print Dialog Box

The Print dialog box contains the following options:

Name

Allows you to choose a printer.

Properties button

Brings up a dialog box where you can choose a printer from your installed printers and specify paper size, source, and orientation (landscape or portrait).

Print to file

Directs your output to a file rather than to a printer.

All

Prints all pages in the worksheet.

Pages

Prints a range of pages. Fill in the "from" and "to" boxes to indicate your selection. If you've selected a plotter as your output device, you can only print one page.

Selection

Prints only the selected regions. If the button is unavailable, you have not selected any regions.

Print Current Page

Prints only the active page. The active page is the one on which the cursor is located, and is indicated in the Status Bar. This may not necessarily be the page visible in the window.

Number of copies

If this option is unavailable, your printer may not support this option.
