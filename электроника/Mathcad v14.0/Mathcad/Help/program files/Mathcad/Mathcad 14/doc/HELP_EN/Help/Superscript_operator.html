Column Operator

Keystroke: [Ctrl] 6

M<n>

Returns the nth column of the array M. If M has no nth column, you'll see an appropriate error message.

Operands:

  • M is an array.
  • n is a dimensionless integer. The acceptable range of n will depend on the size of M and the ORIGIN of the worksheet.

Notes:

  • To get the nth row of a matrix, use the column operator in conjunction with the transpose operator.
