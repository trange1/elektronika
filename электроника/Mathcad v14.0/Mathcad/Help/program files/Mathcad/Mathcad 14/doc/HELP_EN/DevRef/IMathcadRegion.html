Region Class

The Region class represents a Mathcad region within a Worksheet object. It encompasses the following properties:

Properties

  • Tag: gets the region's tag content
  • X: gets the region's left-edge horizontal position
  • Y: gets the region's top vertical position
  • Type: gets the region's type enumeration (text, math, and so on)

The Region class has been extended with the Region2 class with properties for inspecting and setting math region XML, units, and error messages, and determining the selected region. If you are scripting, address the Region2 properties using the Region object.
