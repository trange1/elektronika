Format Menu Commands

Equation

Displays a list of available math styles. To change the font of an existing math style, select the style and click "Modify."

Result

Opens the Result Format dialog box for controlling how numerical results are displayed in the worksheet. Double-click on a specific result to format that result only.

Text

Opens the dialog box for changing the font, size, color, and style of selected text.

Paragraph

Opens the dialog box for changing the text paragraph properties such as alignment, indentation, bulleting, and numbering.

Tabs

Opens the Tabs dialog box to set tab stops. You can also control tab stops using the worksheet ruler.

Style

Opens the Text Styles dialog box to create or modify a text style. If you first click on a text region, the dialog box applies a text style to only that region.

You can apply a text style simply by choosing a style from the left-most drop-down list in the Formatting toolbar.

Properties

Provides formatting options to a region or group of regions.

Graph

Provides different formatting options for graphs.

Color

Formats the background, highlight, and annotation colors as well as color palettes for images.

Area

Locks, unlocks, collapses, and expands areas in the worksheet.

Separate Regions

Separates any overlapping regions in your worksheet.

Align Regions

Aligns selected regions either across (horizontally by top borders) or down (vertically by left borders).

Repaginate Now

Rearranges "soft" page breaks so that they do not intersect regions in the worksheet. Be aware that any uncalculated regions will calculate when you choose this command. Mathcad automatically repaginates before printing or previewing a worksheet.
