Pasting Content into Mathcad

Mathcad pastes the following way:

  1. Mathcad regions remain as Mathcad regions.
  2. Contents from another Windows application pasted in a blank area of the worksheet become OLE objects.
  3. An array of numbers or strings pasted into an equation placeholder come in as a matrix.
  4. Text from another application pasted into a text region remains as plain text.

If you are dissatisfied with the way an object pastes into Mathcad, try the Edit > Paste Special option instead.
