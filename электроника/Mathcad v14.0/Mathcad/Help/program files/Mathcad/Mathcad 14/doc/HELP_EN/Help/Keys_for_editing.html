Keys for Editing Text

[Enter] Insert blank line. In text, begin a new paragraph.

[Delete] Delete blank line. In text or math, remove character to the right of the insertion line.

[Shift] [Enter] In text, begin a new line within a paragraph.

[Ctrl] [Enter] Insert a page break.
In text, set the width of the text region.
In math, insert addition with linebreak operator.

[Ctrl] A In text, select all the text in the text region.

[Ctrl] [Shift] [Enter] In a region, move the cursor out of and below the region.

[Ctrl] F Find a string of characters.

[Ctrl] H Replace a string of characters.

[Ctrl] Z Undo last edit, up to 100 actions.

[Ctrl] Y Redo. Reverses the action of Undo, up to 100 undos.

[Ctrl] C Copy selection.

[Ctrl] V Paste copied or cut selection into worksheet.

[Ctrl] X Cut selection.

[Insert] Start insert mode, toggle to start overwrite mode.

[Ctrl] K Insert or edit hyperlink.

Key Movement in Text

Enter Key

Starts a new line in a new paragraph.

Space Bar

Inserts a space.

Tab Key

Moves the text cursor to the next tab stop on the right; any text to the right of the text cursor is moved accordingly. At the end of the text region, the region is extended to the next tab stop.

[Shift] [Tab] Inserts a space.

Home Key

Moves the text cursor to the beginning of the line.

[Ctrl] [Home] Moves the text cursor to the beginning of the first line of text in the current text region.

End Key

Moves the text cursor to the end of the line.

[Ctrl] [End] moves the text cursor to the end of the last line of text in the current text region.

Left Arrow

Moves the text cursor one character to the left. At the beginning of a text region, the cursor moves into the empty space to the left.

[Ctrl] Left Arrow Moves the text cursor to the beginning of the previous word.

[Shift] Left Arrow Selects the next character to the left of the text cursor.

Right Arrow

Moves the text cursor one character to the right. At the end of the text region, the cursor moves into the empty space to the right.

[Ctrl] Right Arrow Moves the text cursor to the end of the next word.

[Shift] Right Arrow Selects the next character to the right of the text cursor.

Up Arrow

Moves the text cursor to the previous line. At the top of the text region, the cursor moves into the empty space above.

[Ctrl] Up Arrow Moves the text cursor to the beginning of the current line. At the beginning of the line, the cursor moves to the beginning of the previous line. At the top of the text region, the cursor moves into the empty space above.

[Shift] Up Arrow Moves the text cursor one line up and selects all text between the old and the new locations of the cursor.

Down Arrow

Moves the text cursor to the next line. At the last line, the cursor moves into the empty space below.

[Ctrl] Down Arrow Moves the text cursor to the end of the current line. At the end of the line, the cursor moves to the end of the next line. At the end of the last line, the cursor moves into the empty space below.

[Shift] Down Arrow Moves the text cursor one line down and selects all text between the old and the new locations of the cursor.
