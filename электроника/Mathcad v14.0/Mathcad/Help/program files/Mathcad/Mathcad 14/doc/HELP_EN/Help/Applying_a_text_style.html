Applying a Text Style

When you create a text region, the "Normal" text style is used. However, you can apply other text styles, to any paragraph of text you create.

To apply a text style, first you must click in a paragraph or select all the text in multiple paragraphs.

There are several options you can use:

  1. Choose a style from the left-most drop-down list in the Formatting toolbar, or

  2. Choose Style from the Format menu or right-click and choose Style to open the Text Style dialog box.

  3. Select one of the available text styles and click "Apply." The text in your region acquires the text and paragraph properties associated with that style.

Notes:

  • When you apply a text style to a text region, the region's paragraph properties change to those of the applied text style.

  • If you have previously formatted text by manually changing the font properties, a dialog appears asking if you want to "Reset all formatting based on selected style." If you click yes, everything in the selected paragraph changes to the new style. If you click no, font properties remain. Be careful because you might lose symbol fonts set for Greek characters, text you have made bold, and so on.
