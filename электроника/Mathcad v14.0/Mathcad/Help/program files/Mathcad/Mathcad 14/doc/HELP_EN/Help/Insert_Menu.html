Insert Menu Commands

Graphs

Displays a submenu listing two-dimensional and three-dimensional graphs available in Mathcad.

Matrix

Opens the Insert Matrix dialog for creating a vector or matrix. If the cursor is in a vector or matrix, you can insert or delete rows or columns.

Function

Opens the Insert Function dialog with a scrolling list of built-in functions. Click on an individual function to see its arguments, description, or to insert it into your worksheet.

Unit

Opens the Insert Unit dialog with a scrolling list of built-in units. Select a unit to insert it into the unit placeholder to the right of a result.

Picture

Inserts the picture operator containing a placeholder for specifying a variable name or file.

Area

Inserts area boundaries into your worksheet.

Page Break

Inserts a hard page break wherever the cursor is located.

Math Region

Creates a math region inside text. You can type an expression, variable, function, or other math in the placeholder.

Or creates a math region in a blank part of your worksheet.

Text Region

Creates an empty text region wherever the crosshair is. The crosshair changes into a red vertical editing line.

Component

Opens the Insert Component dialog. Components allow data to flow dynamically between Mathcad and a data source or application.

Data

Allows you to import or export data to different applications with a variety of options.

Control

Inserts the selected control component into the worksheet. Controls allow you to control a Mathcad calculation without using a variable or function definition, and are customized using a scripting language such as VBScript or JScript. You can also use the Control toolbar.

Object

Brings up dialog box with a list of OLE applications registered on your system.

Inserts an OLE object into your worksheet. You can create a new object, insert a copy of an existing file, or link to an existing file.

Reference

Opens the Insert Reference dialog, allowing you to access definitions in a saved worksheet.

An icon appears displaying the full path of the referenced file. Definitions from this file become available anywhere to the right of and below this icon.

Hyperlink

Brings up the Insert Hyperlink dialog allowing you to create, edit, or remove a hyperlink.
