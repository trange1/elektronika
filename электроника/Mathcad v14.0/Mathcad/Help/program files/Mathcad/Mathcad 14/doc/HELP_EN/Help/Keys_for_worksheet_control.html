Keys for Worksheet and Window Management

[Ctrl] N Create new worksheet.

[Ctrl] O Open worksheet.

[Ctrl] P Print worksheet.

[Ctrl] S Save worksheet.

[Ctrl] [F4] Close worksheet.

[Alt] [F4] Quit.

[Ctrl] R Redraw screen.

[Ctrl] [F6] Make next window active.

[F1] Open Help.

[Shift] [F1] Enter or exit context sensitive Help.

[Esc] Exit context sensitive Help or interrupt a calculation.

Enter Key

Moves the crosshair one space down and all the way back to the left edge of the worksheet.

Spacebar

Moves the crosshair one space to the right.

Tab Key

Moves the crosshair to the right, to the next tab stop.

Home Key

Moves the crosshair beside the previous region.

[Ctrl] [Home] Moves the crosshair beside the first region in the worksheet.

End Key

Moves the cursor to the next region.

[Ctrl] [End] moves the cursor to the last region in the worksheet.

Left Arrow

Moves the crosshair cursor one space to the left. The cursor enters a region to the left.

Right Arrow

Moves the cursor one space to the right. The cursor enters a region to the right.

Down Arrow

Moves the cursor down one space. The cursor enters a region below.

Up Arrow

Moves the cursor up. The cursor enters a region above.

Page Up

Moves up about 80% of the window height.

[Shift] [Page Up] Moves up to the previous page break.

Page Down

Moves down about 80% of the window height.

[Shift] [Page Down] Moves down to the next page break.

Nudging Regions

Left Arrow

Moves the selected region(s) one space to the left.

Right Arrow

Moves the selected region(s) one space to the right.

Up Arrow

Moves the selected region(s) one space up.

Down Arrow

Moves the selected region(s) one space down.

Note:

  • If you hold the arrow keys down, the regions move more quickly.
