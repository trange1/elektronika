File - Save As Web Page

Prompts you for a new name under which to save the current worksheet with a .HTM extension. The current worksheet name is the default. Once you click Save, you get a second dialog of preferences, including:

  • Format to use for images (.jpg, .png)

  • Template wrapper file, if desired

  • Page title, if desired

  • Option to open the saved page immediately in your default browser.

Mathcad worksheets are exported as HTML text with appropriate markup. Equation regions are saved as images.
