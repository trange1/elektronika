Adding Hyperlinks for HTML Export

Hyperlinks allow users to quickly browse the contents of your Mathcad worksheets. Mathcad hyperlinks are exported as .HTM anchor links in standard HTML markup. Be sure to export all linked and referenced worksheets to ensure that these links point to existing .HTM files.

Note that in order for hyperlinks to export correctly to HTML, you must check the "Use Relative Path" box on the Insert Hyperlink dialog in Mathcad. Since HTML links are usually displayed as colored and underlined, they look better if a meaningful word or phrase is selected for the link.

After creating the Mathcad worksheets, aligning them properly, linking them together, and creating a table of contents, create an HTML Template file to specify any HTML wrapping for your exported documents.
