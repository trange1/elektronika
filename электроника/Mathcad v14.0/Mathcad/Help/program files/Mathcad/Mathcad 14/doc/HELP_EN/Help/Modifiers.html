Modifiers

Modifiers are special words that you can type after a keyword to modify the operation of the keyword. The following describes the available modifiers and lists the keywords you can use them with.

Modifier Purpose Keywords That
Use the Modifier
ALL Apply keyword to each and every variable in an expression. assume, explicit
acos Rewrite expressions in terms of the inverse cosine function. rewrite
acot Rewrite expressions in terms of the inverse cotangent function. rewrite
asin Rewrite expressions in terms of the inverse sine function. rewrite
atan Rewrite expressions in terms of the inverse tangent function. rewrite
cauchy Return the Cauchy principal value of an integral. Symbolic evaluation of integrals
complex Specify that a variable is a complex number, or that an operation is performed over the complex numbers. assume, factor, parfrac
cos Rewrite expressions in terms of the cosine function. rewrite
cosh Rewrite expressions in terms of the hyperbolic cosine function. rewrite
cot Rewrite expressions in terms of the cotangent function. rewrite
coth Rewrite expressions in terms of the hyperbolic cotangent function. rewrite
degree Return a second column in the output of "coeffs" containing the degrees of the terms. coeffs
domain Specify the domain or set of input values of a variable. factor, parfrac
even Specify that a variable is an integer divisible by two. assume
exp Combine or rewrite expressions using identities for the exponential function. combine, rewrite
fraction Return a continued fraction as a fraction. confrac
fully Return a detailed solution to an equation. All - see Special Modifiers
gamma Rewrite expressions involving factorials in terms of the gamma function. rewrite
integer Specify that a variable is an integer. assume
ln Combine or rewrite expressions using identities for the natural logarithm function. combine, rewrite
log Combine or rewrite expressions using identities for the base-10 logarithm function. combine, rewrite
matrix Return a continued fraction in array form. confrac
max Perform additional steps of the "simplify" algorithm for greater simplification. simplify
odd Specify that a variable is an integer not divisible by two. assume
raw Return results in unsimplified form. fourier, invfourier, laplace, invlaplace, ztrans, invztrans, substitute
real Specify that a variable is a real number, or that an operation is performed over the real numbers. assume, factor
RealRange Specify that a variable is in a range of real numbers. assume
signum Rewrite expressions involving the Heaviside function in terms of the signum function, which extracts the sign of a number. rewrite
sin Rewrite expressions in terms of the sine function. rewrite
sincos Combine or rewrite expressions using identities for sine and cosine. combine, rewrite
sinh Rewrite expressions in terms of the hyperbolic sine function. rewrite
sinhcosh Combine or rewrite expressions using identities for hyperbolic sine and hyperbolic cosine. combine, rewrite
tan Rewrite expressions in terms of the tangent function. rewrite
tanh Rewrite expressions in terms of the hyperbolic tangent function. rewrite
using Replace a generated variable in the solution to an equation. All - see Special Modifiers

Special Modifiers

The modifiers "fully" and "using" can be used as keywords on their own, without following other keywords. You can also use them in combination with other keywords on a separate line. See Quicksheet: Symbolic Solutions of Equations for examples.
