Insert Unit Dialog Box

Dimension

Lists quantities of built-in units. Only units in the selected dimension or quantity are displayed in the Unit list. Choose "All" to see all built-in units in the current unit system.

System

Shows the default system of units. To change the system of units, choose Worksheet Options from the Tools menu and click on the Unit System tab.

Unit

Displays a list of units available for the selected quantity. To see the complete list of units, choose "All" in the "Dimension" box.

Double-click on a unit to insert it into your worksheet, or click the Insert button.

Insert

Inserts the unit into your worksheet. If the cursor is in a unit placeholder in your worksheet, the dialog attempts to display the correct quantity associated with the current result.
