E-books

As described in the Introduction to Mathcad E-books, an E-book is a hyperlinked collection of Mathcad worksheets. Each Mathcad E-book opens in its own window. An E-book has a table of contents and an index, as well as other browsing features that you can access using the toolbar buttons in the window. The worksheets in an E-book are live, so a reader can experiment directly within the book.

If you have several Mathcad worksheets that you want to collect together, you can create your own E-book. The steps to creating an E-book include:

  1. Creating individual Mathcad files
  2. Preparing a Table of Contents
  3. Adding hyperlinks between files
  4. Creating an HBK file to specify the order of the files in the book
  5. Developing an index
  6. Checking the index, HBK file, and worksheets for errors

Each step is detailed in the accompanying sections to demonstrate the creation of a sample E-book called "Explorations in Algebra." We recommend that you read all steps of the process before starting.
