Exponential Distribution

Distribution generators for the Exponential Equation:

r·exp(-r·x)

dexp(x, r) Returns the probability density for value x.

pexp(x, r) Returns the cumulative probability distribution for value x.

qexp(p, r) Returns the inverse cumulative probability distribution for probability p.

rexp(m, r) Returns a vector of m random numbers having the exponential distribution.

Arguments:

  • x is a scalar or vector of real values greater than or equal to 0. To allow integration and other operations over this argument, values outside of the stated range are allowed, but produce a 0 result.
  • r is a real rate, greater than 0.
  • p is a real probability between 0 and 1.
  • m is an integer greater than 0.

Notes:

  • For definitions of the functions, and information on generating random numbers, see the Density and Distribution definitions.
