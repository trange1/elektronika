Importing a Graphic

Importing an Image by Copying and Pasting

To copy and paste an image into Mathcad from another application:

  1. Select and copy the image in the source application.

  2. In Mathcad, place the crosshair where you want to the image to appear.

  3. Choose Paste or Paste Special from the Edit menu to paste the image into the worksheet.

Notes:

  • When you choose Paste, the image is pasted according to the default information stored with it. If you want to paste the image as a particular kind of object (bitmap, linked object, or embedded object), choose Paste Special.
  • If you wish to manipulate your image mathematically, insert it as a Picture.

Importing an Object by Dragging and Dropping

To import a linked object directly from another application:

  1. Open the source application.

  2. Arrange your Mathcad window and the source application window side-by-side on the screen.

  3. Select the graphic in the source application.

  4. Hold the mouse button down and drag the cursor to the worksheet.

  5. Release the mouse button to drop the graphic into the worksheet.

Editing a Linked Object

To edit a graphics image after inserting or pasting it into your worksheet:

  1. Double-click on the graphic.

  2. If the graphic is from an application that supports in-place activation, a hatched border surrounds the graphics object and the menus and toolbars change to the source application's menus and toolbars. If the graphic is from an application that does not support in-place activation or if the graphic is a linked object, a separate window for editing appears.

  3. Make any necessary edits.

Note:

  • If you insert a bitmap using Insert > Picture or if you paste an image as a bitmap, you cannot edit it using the steps outlined above.

Resizing a Graphics Region

To change the length and width of a graphics region, select it, then drag on the black handles to change its length and width.

To restore a resized bitmap to its original size:

  1. Click on the image with the mouse button.

  2. Choose Properties from the Format menu and click on the Display tab.

  3. Click on the box labeled "Display at original size."

  4. Click "OK."
