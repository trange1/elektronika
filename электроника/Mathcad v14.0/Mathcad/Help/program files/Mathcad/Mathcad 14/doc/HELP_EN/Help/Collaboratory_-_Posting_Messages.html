Posting Messages in the User Forums

To post a new message or a reply to an existing one:

  1. Click on the forum name to show the messages under it.

  2. To post a new message, choose Post from the menu bar at the top of the User Forums window. To reply to a message:

    • Click on the message.

    • Click "Reply" at the top of the message screen.

  3. Enter the title of your message in the Topic field.

  4. Click on the boxes below the title if you want to attach a file, or preview and spell check your message.

  5. Type your text in the message field. You can include hyperlinks in your message by entering an entire URL such as http://www.myserver.com/main.html or other HTML code if you wish.

  6. Click "Post" at the bottom or top of the message window.

  7. If you are attaching a file, a new page appears. Specify the file type and file then click "Upload Now."

To delete a message that you posted, click on it to open it then click "Delete" in the small menu bar just above the message.

For more information on posting or reading messages and other features of the Mathcad User Forums, click Help on the User Forums menu bar.
