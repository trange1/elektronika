Format > Properties

Brings up the Properties dialog box. You may choose the following options, depending on the kind of region(s) selected:

  • Highlight a region in a color you choose.
  • Insert or remove a border around a picture.
  • Display a picture at its original size.
  • Turn a calculation on or off by clicking the "Disable Evaluation" button.
  • Turn Optimization on or off.
  • Set text region properties to control the width and behavior of a text region.
  • Protect a region from editing when the worksheet is protected.

Notes:

  • You can select multiple regions in your worksheet, including both math and text, and set common properties for those regions using the Properties dialog box.
  • You can set any region's properties by right-clicking on that region.
