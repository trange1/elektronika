User-Defined Functions

You can extend Mathcad's power by writing your own customized functions, by writing a DLL in C or C++ that uses our UserDLL interface.

Functions written using the UserDLL interface can accept and return complex scalars and arrays, as well as strings. Customized functions can also have features such as error messages, interruption, and exception handling. To create UserDLL functions, you first need to create source code using the structures and functions provided by PTC. Next, you compile the code and link it to create a DLL.

For information on creating DLLs, such as the structures, functions, procedure, and other details, refer to the Developer's Reference. You should also check www.ptc.com/go/mathsoft/mathcad/ for updated information.
