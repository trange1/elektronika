WorksheetMetadata Class

The WorksheetMetadata class is a child interface of the Mathcad Metadata class. In addition to inheriting the properties and methods of its parent, it contains the following properties:

Properties

  • Author
  • Company
  • Description
  • Keywords
  • Revised By
  • Title
  • Document identifier
  • Version identifier
  • Branch identifier
  • Parent Version identifier
  • Revision Number
