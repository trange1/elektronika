Equal Sign for Evaluation

Keystroke: [=]

x = ...

Returns the numerical evaluation of x as a result on the right-hand side. If the expression on the left has not yet been defined or has some other syntax or evaluation problems, an appropriate error message appears.

Operands:

  • x is any valid Mathcad expression or variable name.

Notes:

  • You cannot directly edit what appears to the right of the [=]. You can, however, change the format or units in which the numbers are displayed.
  • Certain symbolic functions are not defined for normal Mathcad evaluation, only for symbolic evaluation, and produce errors when used with [=].
