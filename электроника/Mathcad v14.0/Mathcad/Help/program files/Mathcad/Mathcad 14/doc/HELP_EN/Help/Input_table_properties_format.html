Data Table Properties: Format

Displayed Precision

Choose or type an integer. Determines the number of total digits up to 17.

Show trailing zeros

Click to show trailing zeros for results with as many digits to the right of the decimal point as required by the current choice of Displayed Precision.

For example, if Displayed Precision = 3, then 3/2 displays as 1.5 if "Show trailing zeros" is unchecked, and 1.500 if "Show trailing zeros" is checked.

Exponential threshold

Choose or type an integer n. Values smaller than 10−n or greater than 10n are displayed in exponential notation.

Complex threshold

Choose or type an integer m. Controls how much larger the real or imaginary part of a number must be before display of the smaller part is suppressed. Difference in scale of more than 10−m displays only the larger part. For example, the number 1 + 0.002i would display as 1 with complex threshold smaller than 3.

Show row/column labels

Click this box to display the row and column numbers associated with the values in the table.

Font

Brings up a dialog box for controlling the font of the displayed numbers.
