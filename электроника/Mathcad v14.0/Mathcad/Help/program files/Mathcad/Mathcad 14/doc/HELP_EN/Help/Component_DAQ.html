Data Acquisition Component

The Data Acquisition Component (DAC) is not available in Mathcad 14.

Mathcad 14 cannot create new Data Acquisition Components. However, worksheets that already contain Data Acquisition Components will calculate properly with Mathcad 14.

PTC anticipates that support for existing Data Acquisition Components will cease in a future release. If you want or need to import data into Mathcad, you should use either the Data Table, the Data Import Wizard, or one of Mathcad's file access functions.
