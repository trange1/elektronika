File Input and File Output: Options Tab

Select Data > File Input or Data > File Output from the Insert menu.

File Options

Specify the file format, filename, and path.

Data Range

Specify only certain rows or columns of a data file. For example, you can leave out label and title rows when reading in data.

  • Start at Row: Type in the number of the row containing the first value to read.

  • Start at Column: Type in the number of the column containing the first value to read.

By default, "Stop at End of Data" is turned on for both rows and columns. This option makes the File Input and File Output components read all the values from the starting value to the end of the file.

Alternatively, uncheck "Stop at End of Data" to turn it off and enter a value in the "Stop at Row" box and/or the "Stop at Column" box.
