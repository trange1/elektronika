Saving a Worksheet as HTML

To save a worksheet in HTML format, first set Mathcad's HTML Options. Once you've selected image and layout formats, do the following:

  1. Choose Save As from the File menu.

  2. In the "Save as type" drop-down list, choose "HTML File" and provide a file name.

  3. Click "Save."

The file is saved as an .HTM file. Text is saved as text with HTML markup. All graphs and other images are saved in either JPEG or PNG format.

If you have included an HTML template in your Preferences, the template regions are only visible in the browser window.

Notes:

  • The .HTM file can not be read back into Mathcad.

  • When you save a worksheet as an HTML document, a subdirectory with the name "(filename)_images" is created to contain associated JPEG or PNG files.

  • You can also save worksheets in HTML format by choosing Save as Web Page from the File menu, which allows you to save preferences at the time the page is saved.
