Array Subscript Operator

Keystroke: [

v[n

Returns the nth element of v. If v has no nth element, you see an appropriate error message.

M[m,n

Returns the element in row m and column n of M. If M has no mnth element, you'll see an appropriate error message.

Operands:

  • v is a column vector.
  • M is an array.
  • m and n must be dimensionless integers. The acceptable range of m and n depends on the size of v or M and the ORIGIN of the worksheet.

Notes:

  • You can use the subscript operator to assign individual vector or array elements, rather than using the Matrix dialog. If used with range variables, it assigns an entire vector or array.
  • You can create an m × n matrix by defining the bottommost corner element with the subscript operator, for example

    M[100,50 := 0

    Unspecified elements are filled with 0's. The expression above creates a 101 × 51 element matrix (assuming ORIGIN of 0), entirely filled with 0s. Be careful of inadvertently creating large arrays, or arrays with unexpected 0 elements by defining elements individually out of sequence.

  • If you wish to access an entire column of your array, use the column operator.
