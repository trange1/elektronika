Sum with Range Variables and Vectors

Keystroke: [$]

Returns the sum over i of the expression X.

Keystroke: [Ctrl] 4

Σ v

Returns the sum of all the elements in the vector v.

Operands:

  • X is any expression producing a value for each element in i.
  • i is a range of integers. The acceptable range depends on the size of X and the ORIGIN of the worksheet.
  • v is a vector.

Notes:

  • Define the range variable, i, before using the sum with range variable operator. To include the range variable as part of the operator, use the sum without range variables operator.
  • If the summand has several terms, be sure to enclose them in parentheses to keep them inside the summation operator.
  • Under symbolic evaluation, if no range is specified, results are typically based on ranges which start at 0 or 1.
