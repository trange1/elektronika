Evaluation Toolbar

Operators used to evaluate expressions numerically or analytically, and to create new, custom operator definitions, are found on this toolbar. Click on the button in the Math toolbar to bring up the Evaluation toolbar, or choose Toolbars > Evaluation from the View menu.

Evaluation Operators

  • evaluation
  • definition
  • global definition
  • live symbolic evaluation
  • symbolic keyword evaluation
  • custom prefix operator
  • custom postfix operator
  • custom infix operator
  • custom treefix operator

Tooltips

To learn what a button does, hover the cursor over the button until a tooltip shows the title and a description appears on the message line of the Status Bar.

Note that the tooltips are only documented for US keyboards. The Mathcad Keyboard Shortcuts page may show the shortcuts for your language keyboard.
