Edit - Replace

Allows you to replace one set of characters with another.

Replace Dialog Box

The Replace dialog box has the following options:

Find what

Enter the characters to find.

Replace with

Enter the characters to replace those in the "Find what" box.

Match whole word only

Locates a whole word rather than part of a word.

Match case

Locates characters having the same case as those you enter (lowercase or uppercase).

Find in Text Regions

Locates characters in text regions only.

Find in Math Regions

Locates characters in math regions only.

Find in Collapsed Areas

Locates characters in collapsed areas.

Find Next

Locates the next or previous occurrence of the characters.

Replace

Substitutes the characters in the "Replace with" box for the currently selected characters.

Replace All

Substitutes the characters in the "Replace with" box for all occurrences of the characters in the "Find what" box.
