Previewing Your Worksheet

To see how your worksheet will look on the page before printing:

  1. Choose Print Preview from the File menu. The Print Preview screen shows the current page of your worksheet as it will appear when printed.

  2. Page through your worksheet using the "Next Page" and "Prev Page" buttons at the top of the screen.

  3. Click "Close" to return to the main worksheet window.

To magnify the worksheet move the cursor over the page until the cursor changes to a magnifying glass, then click the mouse. Click again to magnify your worksheet even more. Once you've reached the maximum magnification, clicking on the page de-magnifies it.

If you want to make changes to the layout, choose Page Setup from the File menu to change margins or add headers and footers.
