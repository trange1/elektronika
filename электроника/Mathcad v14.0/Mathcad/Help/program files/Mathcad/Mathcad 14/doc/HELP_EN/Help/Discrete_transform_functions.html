Fourier and Wavelet Transform Functions

fft, ifft, FFT, IFFT Return the Fast Fourier Transform and Inverse Transform of real-valued data in vector v. These functions assume vectors of length 2m.

cfft, icfft, CFFT, ICFFT Return the Discrete Fourier Transform and Inverse Transform of real or complex-valued data in vector or array A of arbitrary size.

wave, iwave Return the wavelet transform and inverse transform assuming vectors of length 2m.
