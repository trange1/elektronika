Number of Elements in a Vector

length(v) Returns the number of elements in the vector v.

last(v) Returns the index of the last element in the vector v.

Arguments:

  • v is a column vector.

Notes:

  • The length of a vector is always a measure of the number of entries, while the index of the last element is a function of the ORIGIN variable. You may find the last function to be a convenient tool for setting the length of program loops.
  • To get the Euclidean length of v, use the magnitude operator.
  • If you need the size of a matrix, rather than a vector, use rows and cols.
