Symbolic Functions

The following symbolic functions can only be evaluated using the symbolic equal sign or the Symbolics menu. They cannot be evaluated using the equal sign for evaluation (=).

Note: To return numerical results for these functions, use the keyword float.

beta(x,y) - Beta function

Chi(x) - Hyperbolic cosine integral function

Ci(x) - Cosine integral function

denom - Returns the denominator of a fraction or rational expression.

dilog(x) - Dilogarithm function

Δ(x) - Dirac delta (unit impulse) function

Δ(n,x) - nth derivative of the Dirac delta function

Ei(x) - Exponential integral function (Cauchy principal value) for a real number x

Ei(n,x) - Generalized exponential integral function for an integer n and complex number x. −Ei(1, −x) returns the complex exponential integral function. Note that Ei(x) = −Re(Ei(1, −x)).

fact2(n) - Double factorial function

FresnelC(x) - Fresnel cosine integral function

FresnelS(x) - Fresnel sine integral function

hypergeom(n,d,x) - Hypergeometric function

LambertW(x) - Lambert's W function

LambertW(n,x) - nth branch of Lambert's W function. Note that LambertW(x) = LambertW(0,x).

numer - Returns the numerator of a fraction or rational expression.

polylog(n,x) - Polylogarithm function

Psi(x) - Digamma function

Psi(n,x) - Polygamma function. Note that Psi(x) = Psi(0,x).

Shi(x) - Hyperbolic sine integral function

Si(x) - Sine integral function

Zeta(s) - Riemann Zeta function
