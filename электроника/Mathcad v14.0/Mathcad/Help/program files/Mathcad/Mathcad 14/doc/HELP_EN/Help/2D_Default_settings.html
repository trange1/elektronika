Default Settings for 2D Plots

To reset a specific plot to the default settings:

  1. Double-click on the plot to open a tabbed dialog box.

  2. Click on the Defaults tab.

  3. Click the "Change to Defaults" button.

Copying defaults from an existing graph

To make the default settings be those of a particular plot, follow the same steps, but check the "Use for Defaults" check box.

To set plot defaults for the whole worksheet

  1. Make sure that you don't have any graphs selected.

  2. Choose Graph > X-Y Plot or Graph > Polar Plot from the Format menu to open the tabbed dialog box.

  3. Change the desired settings on the X-Y Axes and Traces pages.
