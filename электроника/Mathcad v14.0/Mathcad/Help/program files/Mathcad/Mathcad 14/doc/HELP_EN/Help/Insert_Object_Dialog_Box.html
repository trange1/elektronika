Insert Object Dialog Box

The Insert Object dialog box has the following options:

Create New

Inserts an object (graphic, movie, other media) by opening the application. In this case the "Object Type" listbox lists the available applications in which to create a new object. This option creates an embedded object.

Create from File

Inserts an object from a file. In this case, the "File" text box allows you enter a file name, and a "Browse" button is available to locate a stored file.

Link

Links an object from an OLE server into your worksheet. When you make changes to a linked graphic, you change the original version. "Create from File" must be selected for this option to be available.

Display as Icon

Displays the object as an icon.
