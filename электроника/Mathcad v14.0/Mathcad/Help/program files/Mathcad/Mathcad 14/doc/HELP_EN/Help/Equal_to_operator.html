Equal to Operator

Keystroke: [Ctrl] [=]

x = y

Returns 1 if x = y, 0 otherwise.

Operands:

  • x and y are real or imaginary scalars, vectors, or strings.

Notes:

  • You can change the display of this operator to non-bold equals.
  • The evaluation of this operator is controlled by a setting in Worksheet Options from the Tools menu, allowing you to choose exact or fuzzy equality in your calculations.
  • This operator is useful for defining expressions in Mathcad that you don't wish to evaluate, as it does not yield a value until an = operator is applied to the end.
  • Boolean equals are used to define equations in solve blocks.
