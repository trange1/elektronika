Help Menu Commands

Mathcad Help

Opens Mathcad online Help in the Resources Window.

What's This?

Opens Mathcad's context-sensitive Help system, providing assistance on specific menu items or toolbar buttons.

Developer's Reference

The Developer's Reference is a separate Help resource that includes advanced information on:

  • Customizing Mathcad Control components.

  • Creating a function DLL.

  • Using Mathcad as a Automation server.

Author's Reference

The Author's Reference is a separate Help resource that includes:

  • Developing Mathcad E-books.

  • Saving Mathcad files for the Web.

Tutorials

Opens Mathcad tutorial worksheets, including New Features, a Migration Guide for old worksheets, and introductory lessons for new users.

QuickSheets

Opens a set of worksheets demonstrating applied uses of Mathcad functions, operators, and components.

Reference Tables

Opens a series of worksheets of useful formulas and constants for physics, geometry, and engineering references.

E-books > Open Book

Opens Mathcad E-books in the Resources window. You can browse to an HBK (handbook) file or select from the list of E-books installed in the handbook folder within the Mathcad program folder. Installed E-books are automatically listed on the submenu and in the Resources toolbar for easy access. Click on a title to open the E-book.

User Forums

Opens an online community of users exchanging tips and tricks for getting more out of Mathcad.

Mathcad Web Site

Opens the Mathcad product page on the PTC Web site.

Mathcad Training

Opens the PTC Training Services Web site.

About Mathcad

Shows the version number and product code for your copy of Mathcad, which are necessary for registration.
