Modifying, Creating, and Deleting Text Styles

Modifying a Text Style

To change all the text based on a particular text style or to change a text style for future use, you can modify it:

  1. Choose Style from the Format menu to bring up a dialog showing the currently available styles.

  2. Select the name of the text style you want to modify and click "Modify." You see a dialog box displaying the definitions of that text style.

  3. Click "Font" to modify text formats such as the font, font size, styling, and color.

  4. Click "Paragraph" to modify indentation, alignment, and other paragraph properties.

Any text regions previously created with that text style are modified accordingly.

Creating a New Text Style

  1. Choose Style from the Format menu to open the Text Styles dialog, showing the available text styles.

  2. Click "New."

  3. Name the new style in the "Name" box.

  4. To base the new style on an existing style in the current worksheet or template, select a style from the "Based on" drop-down list.

  5. Click the "Font" button to choose text formats. Click the "Paragraph" button to choose paragraph formatting options.

Your new style now appears in the Style box on the far left of the Formatting toolbar and can be applied to any text paragraph. If you want to use the new text style in future worksheets, save your worksheet as a template. You can also copy the style into another worksheet simply by copying a styled region into that worksheet.

Note:

  • If you base a new text style on an existing text style, any changes you later make to the original style modify the new text style as well.

Deleting a Text Style

  1. Choose Style from the Format menu.

  2. Select one of the available text styles from the list.

  3. Click "Delete."

Although the text style is no longer available, any text in your worksheet whose properties were set by that text style remain unchanged.
