Tools Menu Commands

Spelling

Checks the spelling of all text in the worksheet.

Animation > Record or Playback

Creates, records, and plays back animations of plots and functions.

Protect Worksheet

Safeguards the contents of your worksheet to varying levels. When a worksheet is protected, this menu item becomes Unprotect Worksheet.

Calculate > Calculate Now

Updates only the plots and numerical results immediately visible in the worksheet.

Calculate > Calculate Worksheet

Updates all plots and numerical results throughout the worksheet.

Calculate Now and Calculate Worksheet are usually only necessary when your worksheet is in manual calculation mode. The word "Calc" on the message line of the status bar lets you know when the results need to be updated.

Calculate > Automatic Calculation

Toggles between manual and automatic mode.

If checked, all visible results are automatically updated whenever you make a change in the worksheet. You may find working in manual mode to be noticeably faster.

Optimize > Equation (deprecated)

Optimizes only the selected equation.

Optimize > Show Optimization (deprecated)

Displays a small window that contains the optimized form of the selected equation, if it exists. Mathcad uses this optimized form when computing any result involving the equation.

Debug > Resume

Continues evaluation of a program or region after the pause function is evaluated.

Debug > Interrupt

Halts evaluation of a program or region after the pause function is evaluated. This is equivalent to pressing [Esc], and shows the same error message around the interrupted expression.

Debug > Toggle Debugging

Controls whether or not debugging mode is active, which allows debugging functions to output to the Trace Window.

Disable/Enable Evaluation

Controls whether or not an equation is evaluated in your worksheet. Turning off a definition causes an error in any dependent evaluation.

Trace Error

Brings up the Trace Error dialog, to locate errors originating in regions affecting your evaluation.

License > Borrow

Temporarily acquires a FLEXlm floating license from the available pool on a license server for a specified period of time, allowing you to run Mathcad without access to the license server.

License > Return

Allows you to return a borrowed license before its expiration date.

Worksheet Options

Displays the Worksheet Options dialog box for setting built-in variables, calculation, display, unit system, and dimensions.

Preferences

Displays the Preferences dialog, for customizing aspects of the Mathcad user interface.
