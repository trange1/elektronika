Getting information on a WAV file

GETWAVEINFO(file) Returns a vector containing the format information of a WAV file:

  • The number of channels

  • The sample rate

  • The number of bits per sample (or resolution)

  • The average number of bytes per second that an audio player device would have to process to play the audio in real time.

Arguments:

  • file is a string. Type ", followed by the path and name of the file, then type " again.

Note:

  • file must be a pulse code modulated (PCM) Microsoft WAV file.
