Other Sources of Information

Electronic Documentation

  • Mathcad User's Guide: an introduction to using Mathcad to solve technical problems.
  • Release Notes: last-minute changes to Mathcad functionality, including known incompatibilities, problems fixed in the last few releases, and installation information.

Mathcad Extension Packs

A Mathcad Extension Pack adds new functions to Mathcad and sometimes other components. Documentation and sample files are provided through an E-Book and additional Help files.

Mathcad E-books

The following Mathcad E-Books are available from the Resources toolbar or under the Help menu. The examples are live and can be copied into your worksheets. Once you open a book, click on any topic to open that topic or use the arrow buttons on the toolbar to browse.

  • Online Tutorials: Getting Started Primers, which introduce basic Mathcad concepts; Features In-Depth, for more comprehensive discussion of Mathcad's capabilities; a Migration Guide, which highlights the effect of changes to functionality on worksheets; and New Features, which illustrates functionality new to this release.
  • QuickSheets: working examples of most Mathcad features in problem-solving scenarios.
  • Reference Tables: mathematical formulas and constants.
  • Other E-books and Extension Packs: add-ons that can be purchased in a variety of disciplines, or others that can be downloaded from the Mathcad Web Resources .

Online Help

  • Mathcad User Forums: connects you to a community of Mathcad users.
  • PTC Web site: downloads, registration, and product information.

    Note: The PTC Web site sets a cookie on your computer the first time you access the site. This cookie records your country, to customize site content. If you receive script errors while accessing the PTC Web site in the Resources window:

    1. Open Internet Explorer.
    2. Go to the PTC Web site, http://www.ptc.com.
    3. Select your country from the list provided.

    This should eliminate any script errors when you browse the PTC Web site in the Resources window.

  • Technical Support: solutions to many questions via email, fax and phone, or use the searchable online database.
