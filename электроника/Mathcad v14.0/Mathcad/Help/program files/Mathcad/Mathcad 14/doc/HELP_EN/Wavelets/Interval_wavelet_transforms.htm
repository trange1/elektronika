Interval wavelet transforms

dwti(a,n,f) Returns the n-level interval wavelet transform of the array a by the wavelet filter f.

idwti(a,n,f) Returns the n-level inverse interval wavelet transform of the array a by the wavelet filter f.

Arguments:

  • a is a vector or matrix. If a is a vector, length(a)/2n must be an integer and length(a)/2n must be greater than or equal to m, where symmlet(m) or daublet(m) is being used as f. If a is a matrix, the row and column lengths must satisfy these requirements.
  • n is an integer.
  • f is a vector resulting from daublet(m) (where m is an even integer between 4 and 6, inclusive) or symmlet(m) (where m is an even integer between 4 and 16, inclusive).
