Determining 2D Plot Coordinates

To see values of the coordinates at any point on a trace:

  1. Click in the X-Y plot or Polar plot to select it.

  2. Choose Graph > Trace from the Format menu to open the Trace dialog box.

  3. In the plot region, drag the mouse along the trace whose coordinates you want to see. A dotted crosshair jumps from one point to the next as you drag it along the trace. (To make the crosshair follow your pointer anywhere on the plot, uncheck the "Track Data Points" check box.)

  4. You'll see the coordinates corresponding to the location of the crosshair in the "X-Value" and "Y-Value" and "Y2-Value" or "Radius" and "Angle" boxes. These coordinates change as you move the mouse.

  5. When you release the mouse button, you'll see the coordinates of the last point you selected in the dialog box.

Note:

  • Click the "Copy" button to copy a coordinate value.
