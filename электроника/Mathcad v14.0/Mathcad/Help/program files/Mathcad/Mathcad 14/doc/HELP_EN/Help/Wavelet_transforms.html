Wavelet Transforms

wave(v) Returns the wavelet transform of the data in v using Daubechies four-coefficient wavelet filter.

iwave(u) Returns the inverse discrete wavelet transform corresponding to wave of the data in u.

Arguments:

  • v, u are vectors of length 2m, where m is an integer. Elements of v must be real.

Notes:

  • Wavelet transforms are an alternative to the traditional Fast Fourier Transform.
