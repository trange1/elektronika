Adding Hyperlinks Between Files

You can insert hyperlinks to Mathcad files from each of the topics in your table of contents to allow users to quickly browse the contents of your E-book.

Example:

Explorations in Algebra

Continued Fractions

Searching for Prime Numbers

Journey Around the Square

In this example, you would create hyperlinks from "Continued Fractions," "Searching for Prime Numbers," and "Journey Around the Square" to the files FRACTIONS.XMCD, PRIMES.XMCD, and SQUARE.XMCD, respectively.

After creating the Mathcad worksheets, including the table of contents, you need to create an HBK file to bind the contents together.
