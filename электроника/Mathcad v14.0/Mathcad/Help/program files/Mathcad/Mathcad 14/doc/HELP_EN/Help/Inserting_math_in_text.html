Inserting Math in Text

You can insert a math region into a text region. The math region continues to be live, just as if it were outside text.

  1. Click in the text region where you want to place the math region.

  2. Choose Math Region from the Insert menu or press [Ctrl] [Shift] A to create a math placeholder. You can also right-click on the text region and choose Insert Math Region from the menu.

  3. Type in the math expression just as you would in a math region.

When you're done entering the expression, click elsewhere in the text region to return to typing text.

You can also copy and paste existing math expressions into a math region placeholder in a text region by following the same initial steps.

Note that math definitions inserted in text affect calculations throughout the worksheet. If you want a definition to be purely cosmetic, you can turn it off. You can also customize the display of operators so they appear in publication style.
