Using Help

To access Mathcad Help, choose Mathcad Help from the Help menu or press [F1]. Use the buttons in the Help window to open the navigational controls:

  • The Contents button opens the Table of Contents. Click on each topic to open subtopics within them.

  • The Index button directs you to specific topics. Use the Index if you know what you are looking for.

  • The Search button searches the entire help system for a word or phrase.

Clicking on a topic title opens the topic page in the right-hand side of the window. If you want to increase the viewing area for a topic, you can either resize the window or close the left-hand side by clicking the x in the upper corner.

Tutorials and QuickSheets

Many Help topics contain links to active Mathcad Tutorials and QuickSheets. These live example files enable you to practice and use Mathcad functionality immediately.

Printing a Help Topic

To print a topic in the Help system:

  • Right-click on the topic

or

  • Right-click on the topic in the Table of Contents

then choose Print Topic from the menu.
