Customize Toolbar Dialog

The Customize Toolbar dialog has the following options:

Available toolbar buttons

Displays the buttons that you can add to the toolbar.

Current toolbar buttons

Displays the buttons in the order they appear (from left to right) on the toolbar.

Add

Moves the button selected to the Current toolbar buttons list. You can also drag a button to the list.

Remove

Moves the button selected out of the Current toolbar buttons list. You can also drag a button to the Available toolbar buttons.

Move up

Moves the selected button up one line. On the toolbar, the button shifts left one space. You can also drag a button to a position.

Move down

Moves the selected button down one line. On the toolbar, the button shifts right one space. You can also drag a button to a position.

Reset

Restores the set of buttons as they were when the dialog box was opened.
