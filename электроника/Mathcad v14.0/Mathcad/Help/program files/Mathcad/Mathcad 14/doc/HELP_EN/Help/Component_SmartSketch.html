SmartSketch Component

SmartSketch is a 2D drawing and design tool developed by Intergraph. The SmartSketch component allows you to embed SmartSketch drawings in a Mathcad worksheet and link them to your calculations. For example, your Mathcad equations can determine the size or shape of a drawing. The SmartSketch component makes Mathcad the ideal platform for creating technical illustrations and specification-driven designs.

Inserting a SmartSketch Component

To insert a SmartSketch component into a Mathcad worksheet, you must have one of the following products installed:

  • SmartSketch LE.

  • SmartSketch version 3 or higher.

  • Imagination Engineer.

  • Imagineer Technical version 2.

When you insert the component, you launch the SmartSketch Setup Wizard, which guides you through the configuration process.

On the first page of the Wizard, you can choose to insert an empty SmartSketch drawing into your worksheet, or insert a copy of an existing drawing.

The second page of the Wizard allows you to specify the number of input and output variables for the component, and the corresponding SmartSketch variables for each. If you are using an existing drawing file, the drop-down boxes are populated with the names of the variables, dimensions, or symbols in the drawing. SmartSketch variables are generally defined by dimensions, so if you have not yet dimensioned your drawing, you most likely won't see any variables.

Configuring a SmartSketch Component

The Properties page for the SmartSketch component allows you to change the SmartSketch variables associated with a particular Mathcad input or output. You can also enable Automatic Resizing, which scales the display of the SmartSketch object to fit entirely in your Mathcad window.

To add or remove an input or output variable, right-click on the component itself and select the appropriate entry on the menu. You cannot change the number of inputs or outputs on the Properties page.

Note:

  • Input values that do not have units attached are passed in SI units. For example, if you send 2.0 as input for a length, it is assumed to be 2.0 meters. SmartSketch, by default, converts this to the display units (inches by default) and creates the drawing. Outputs are returned as unitless values.
