Labels Page for X-Y or Polar Plots

To apply labels on an X-Y plot or a Polar Plot,

  1. Double-click on the plot, or choose Graph > X-Y Plot or Graph > Polar Plot from the Format menu to open a tabbed dialog box.

  2. Click on the Labels tab.

To delete existing labels, backspace over them.

Title

Enter a title for the plot. Be sure that "Show Title" is checked if you wish the title to appear in the worksheet.

Above

Makes the title appears above the plot when "Show Title" is checked.

Below

Makes the title appears below the plot when "Show Title" is checked.

Show Title

Check this to display the title entered in the Title text box.

Axis Labels

Enter the text you want to have appear beside each axis in the corresponding text box. Make sure the box beside it is checked. Uncheck this box to suppress display of the axis label.
