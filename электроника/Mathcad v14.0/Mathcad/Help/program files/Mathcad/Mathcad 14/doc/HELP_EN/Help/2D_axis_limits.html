Changing Axis Limits of 2D Plots

To override the default axis limits:

  1. Click in the plot to select it. In an X-Y Plot, you'll see four numbers, one by each axis limit. In the case of X-Y plots with a secondary y-axis, you will see six axis limit placeholders. In a Polar Plot, you'll see two additional numbers on the right of the plot region.

  2. Click on the number closest to the axis limit you wish to change and type a new number or expression.

  3. Click outside the plot region to redisplay the plot using the numbers you specified as axis limits.

Notes:

  • You can also set axis limits by zooming a plot.

  • You cannot override the default angular axis limits. These are always set to 0 degrees and 360 degrees.

  • To return the axis limits to their automatic settings, delete the entry in an axis limit placeholder, leaving the placeholder empty. Click outside the plot region.
