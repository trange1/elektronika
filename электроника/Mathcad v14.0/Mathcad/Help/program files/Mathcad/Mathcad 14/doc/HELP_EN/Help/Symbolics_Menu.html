Symbolics Menu Commands

These menu commands can be used to evaluate expressions analytically instead of numerically. Alternatively, use live symbolics.

Tutorial - Using the Symbolic Menu

Evaluate

Provides symbolic evaluation of expressions in one of the following three manners:

Symbolically

Evaluates algebraic and matrix expressions.

Floating Point

Symbolically evaluates an expression and returns a number of specified precision whenever possible.

Complex

Returns results with distinct real and imaginary parts.

Simplify

Reduces expressions to the simplest terms.

Expand

Expands powers and sums.

Factor

Factors expressions to product, for example, sum of fractions to single fraction.

Collect

Groups terms containing like powers.

Polynomial Coefficients

Finds coefficients for selected polynomial variable.

Variable

Solve

Solves for a variable.

Substitute

Replaces all occurrences of a variable.

Differentiate

Differentiates by the selected variable.

Integrate

Integrates in terms of the selected variable.

Expand to Series

Expands a function or expression in a Taylor or Laurent series around 0.

Convert to Partial Fraction

Generates a partial fraction expansion for an expression.

Matrix

Select a matrix and apply submenu choices:

Transpose, Inverse, Determinant

Transform

Select a variable and perform one of the following symbolic transforms:

Fourier Inverse Fourier
Laplace Inverse Laplace
Z Inverse Z

Evaluation Style

Controls where symbolic results are placed (below, to the right, or in place of) after menu operations.
