Equation format dialog box

The Equation Format dialog box has the following options:

Style Name

Provides a drop-down list of math styles to modify:

  • Variables is the style used for all variables.

  • Constants is the style used for all the numbers.

  • Math Text Font is the style used for graph labels.

  • User1, User2, and so on are styles you can modify, name, and apply to individual constants or variables.

New Style Name

Type a new name here if you want to change the name of the selected style. Usually you only need to change User1, User2, and so on.

Modify

Click to change the font, font style, size, or color of a selected style.

Default Equation Color

Changes the default color for all math styles.
