Context Sensitive Help

You can get specific help on menu commands, toolbar buttons, and error messages in Mathcad.

  • Hover the cursor over menu commands and toolbar buttons to see a tooltip on the message line of the status bar at the bottom of the window.

  • Press [Shift][F1] on a menu command or toolbar button and online Help opens the specific page.

  • Click on error messages, built-in functions, variables, or operators, and press [F1] for a more detailed explanation.
