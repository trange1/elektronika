Creating Mathcad Files

After you have created the necessary Mathcad worksheets for an E-book, place them all in one E-book folder. If there are many worksheets, you can organize them into subfolders. For example, for the E-book titled "Explorations in Algebra," there are three Mathcad worksheets called LINEAR.XMCD, PRIMES.XMCD, and PROBLEMS.XMCD. Store these files in a folder called ALGEBRA. It is also a good idea to place all pop-up worksheets in a subfolder called pop-ups.

To create a consistent look for your worksheets, you can create and use templates and text styles. Also see tips for writing Mathcad worksheets for helpful suggestions on presentation of your work.

Once you have created your worksheets, you can prepare a table of contents for your E-book.

Tip:

  • If you want to include data from a data file in a worksheet, you should insert an input table and import the data into it. This will ensure that the data stays with the file no matter where the file goes. Refer to Mathcad Help for information on importing data to an input table.
