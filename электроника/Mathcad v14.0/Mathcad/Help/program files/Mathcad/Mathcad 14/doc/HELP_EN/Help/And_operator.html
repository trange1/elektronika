Logical Operators

And

[Ctrl] [Shift] 7

x y

Returns 1 if both x and y are non-zero (true). Returns 0 otherwise.

Or

[Ctrl] [Shift] 6

x y

Returns 1 if either x or y are non-zero (true). Returns 0 otherwise.

Exclusive or, xor

[Ctrl] [Shift] 5

x y

Returns 1 if either x or y, but not both, are non-zero (true). Returns 0 otherwise.

Not

[Ctrl] [Shift] 1

x

Returns 1 if x is zero (false). Returns 0 otherwise.

Operands:

  • x and y are real or complex scalars.

Notes:

  • A value of 0 is FALSE, any non-zero value is TRUE. You must follow the Boolean expression with an equal sign for evaluation (=) to get the result of this comparison.
  • Logical operators treat any numeric value that is not 0 as true, including complex numbers.
  • Both and and or first evaluate x; if x is found to be TRUE (non-zero) y is then evaluated for and, and vice-versa for or. This can have the effect of masking errors; for example, the following give different results:

    0 (2/0) = 0 (2/0 is never evaluated)

    (2/0) 0 = 0 gives an error (2/0 causes singularity error)
