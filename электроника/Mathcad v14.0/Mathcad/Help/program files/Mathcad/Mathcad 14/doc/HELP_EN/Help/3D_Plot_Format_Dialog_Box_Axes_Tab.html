3D Plot Format Dialog Box (Axes Tab)

To format the X, Y, or Z axis, click on the appropriate axis tab inside the dialog box.

Draw Lines

Draws grid lines at each grid on the selected axis.

Note: The grid lines are controlled by the options on the Backplanes tab as well. When you set the grid lines for an axis on the Axes tab, you set them for the two backplanes shared by the axis.

Draw Ticks

Draws tick marks at each grid on the selected axis. The color of tick marks depends on the Axis Color.

Note: The tick marks are controlled by the options on the Backplanes tab as well. When you set the tick marks for an axis on the Axes tab, you set them for the two backplanes shared by the axis.

Auto Grid

Chooses the number of grids automatically.

Line Color

Specifies the color of the grid lines.

Number

Specifies the number of grids to use when Auto Grid is not checked.

Line Weight

Specifies the thickness of the grid lines.

Label

When this box is checked, the text specified in the text box appears on the axis.

Show Numbers

Displays numbers at each grid on the selected axis.

Axis Color

Specifies the color of the axis and the tick marks.

Axis Weight

Specifies the thickness of the axis.

Auto Scale

Chooses the axis limits automatically. By default a plot axis is autoscaled.

When this box is checked, autoscale is ON and each axis limit is set to the first major tick mark beyond the end of the data.

When this box is not checked, autoscale is OFF and the axis limits are set right at the data limits.

Minimum Value

Specifies the lower axis limit when Auto Scale is not checked.

Maximum Value

Specifies the upper axis limit when Auto Scale is not checked.

Note:

  • To control how axes are arranged or oriented in your plot, click on the General tab, and use the Axes Style options to specify Perimeter, Corner, or Equal Scales. Click "None" to hide the axes. Changing the rotation of your plot changes which axis projects toward you.
