Rewriting Expressions

To rewrite an expression in terms of elementary functions, select the function and either:

  • Press [Ctrl] [Shift] [.] and type the keyword "rewrite," followed by a comma, in the placeholder.

    - or -

  • Click rewrite on the Symbolic toolbar.

Then type a modifier, specifying the elementary functions to use in rewriting the expression, in the placeholder that appears, and press [Enter].

Modifiers:

The modifiers in the following table specify the functions to use when rewriting the expression:

Modifier Rewrite Expression in Terms of These Functions
acos Inverse cosine
acot Inverse cotangent
asin Inverse sine
atan Inverse tangent
cos Cosine
cosh Hyperbolic cosine
cot Cotangent
coth Hyperbolic cotangent
exp Exponential
gamma Gamma
ln Natural Logarithm
log Logarithm Base 10
signum Signum
sin Sine
sincos Sine or Cosine
sinh Hyperbolic sine
sinhcosh Hyperbolic sine or hyperbolic cosine
tan Tangent
tanh Hyperbolic tangent
