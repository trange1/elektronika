Setting your HTML Output Preferences

To prepare Mathcad documents for export as HTML, set preferences for formatting and images, and select your output template, if desired. Choose Preferences from the Tools menu to show the Preferences dialog box, then click the "HTML Output" tab to set HTML export choices.

Image Export

You can set the format for image export to JPEG or PNG. PNG is a "lossless" format that often results in clearer images for graphs and drawings, while JPEG images are often smaller and usually compatible with older browser versions. All Mathcad objects that are not equations, such as graphs and data tables, are exported as images since they cannot be expressed as HTML.

When you are satisfied with your choices, Save your files as HTML. You can also set these preferences at save time by choosing Save as Web Page from the File menu.

Note:

  • Changes to your template file only affect new Mathcad documents saved as HTML and do not change files already saved.
