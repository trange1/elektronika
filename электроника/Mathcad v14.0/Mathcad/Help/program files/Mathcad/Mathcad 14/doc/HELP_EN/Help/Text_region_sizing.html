Controlling the Width of a Text Region

As you type into a text region, the region continues to grow until it encounters the right margin; the lines then wrap. You can change the width of text regions by several methods.

Resizing a text region

Using the Mouse to Resize a Text Region

  1. Select the text region.

  2. Move the cursor to a handle on the side of the region. It should change to a double-headed arrow.

  3. Drag the mouse to the right or left to change the size of the text region.

Using the Keyboard to Narrow a Text Region

  1. Click in the text region where you want the text region to wrap.

  2. Press [Ctrl] [Enter].

Enlarging a Text Region to the Full Width of the Page

  1. Click in the text region.

  2. Choose Properties from the Format menu.

  3. Click on the Text tab.

  4. Check "Occupy Page Width."

The text region expands to fit the width of the page and moves other regions out of the way.

Note:

  • Once a text region has been created, changing the right margin has no effect on its width.
