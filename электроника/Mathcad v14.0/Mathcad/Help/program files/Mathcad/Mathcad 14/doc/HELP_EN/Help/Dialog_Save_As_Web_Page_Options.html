Save as Web Page Options

These options control how you output HTML pages when using Save as Web Page from the File menu.

The following options are also available on the HTML Options tab in the Preferences dialog box.

Save images as

PNG: PNG is lossless, so while files may be larger, they retain more image fidelity.

JPEG: If you select JPEG, you can also set the compression "Quality." The higher you set this value, the less images are compressed and the larger the resulting image files will be.

Web page template

Choose a template file, if desired. Templates wrap your Mathcad document with additional HTML output. Templates are useful for creating a uniform look and navigation to many Mathcad documents when creating large web sites. See the Author's Reference in the Help menu for additional instructions.

Title

Specify an HTML title tag for your page, which appears in the title bar of the browser when viewing the file.

View saved page in browser

After saving the file, it opens in the default system browser.

Note:

  • With the exception of the "Title" field and the "View saved page in browser" check box, the options in this dialog are the same as those on the HTML Options tab of the Preferences dialog. However, unlike those options, the ones in the Save as Web Page Options dialog are set on a per-file basis and override your selections in the Preferences dialog.
