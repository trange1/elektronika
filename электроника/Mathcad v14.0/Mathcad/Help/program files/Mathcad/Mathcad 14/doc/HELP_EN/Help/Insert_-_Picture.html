Insert > Picture

Choosing Insert > Picture or typing [Ctrl] T inserts the picture operator. The placeholder at the bottom of the operator allows you to insert an image in one of three ways:

  • Type a string containing the name of, or path to, an image file in the placeholder. Supported graphics formats are BMP, GIF, JPG, PCX, and TGA.

  • Type the name of any matrix to display the matrix in 256 shades of gray.

  • Type the names, separated by commas, of three matrices of the same size containing the individual RGB values corresponding to a color image. (You may need to extract color component matrices from your image matrix first. See information on image format in Mathcad.) You see an RGB rendering of the image. You can also display HSV or HLS renderings if your matrices correspond to these color mappings.

Once you display an image with the picture operator, you can make adjustments to its appearance or select information from it using the Picture toolbar and the associated operator Properties dialog box.

Notes:

  • If you modify an image file read by the picture operator, press [F9] to recalculate the region to read in the new image.
  • To read an image into Mathcad as a matrix, use the image file read functions.
  • You can also use Insert > Object to create an embedded or linked graphics object, or you can paste an image in your document, but these options do not create a Mathcad matrix that can be manipulated mathematically.
