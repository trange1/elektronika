Selecting Text

To select a group of characters:

  • Place the cursor at one end of the selection, then drag over to the other end of your selection.

  • When you release the mouse button, the selected text is highlighted.

Other ways to select text:

  • To select one word of text, double-click on the word to highlight it.

  • To select a paragraph, triple-click on a word in the paragraph.

  • To select text character-by-character, place the cursor at one end of the selection, then press [Shift] and an arrow key in the direction you want to select. The highlighted text is selected.

  • To select text line-by-line, place the cursor at one end of the selection, then press [Ctrl] [Shift] and the up or down arrow key.

  • To select text word-by-word, place the cursor at one end of the selection, then press [Ctrl] [Shift] and the left or right arrow key.
