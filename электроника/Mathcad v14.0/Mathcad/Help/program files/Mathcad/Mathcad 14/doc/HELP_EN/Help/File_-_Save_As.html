Save As

Prompts you for a new name under which to save the current worksheet. Gives you the option to save a file as:

  • A Mathcad XML document (*.XMCD)

  • A compressed Mathcad XML document (*.XMCDZ)

  • A previous version (13, 12, 11) of Mathcad (*.MCD, *.XMCD, *.XMCDZ)

  • A Mathcad XML template (*.XMCT)

  • An RTF file, which you can open in Microsoft Word.

  • An HTML file

Use the Save tab on the Preferences dialog to choose a default format.
