Paragraph Format Dialog

Indent - Left, Right

Sets the indentation amount from the left or right side of the text region.

Special

Sets the kind of indent:

  • First Line: Only the first line in the paragraph indented.

  • Hanging: All lines except the first line indented.

By

Sets the indentation amount from the left edge of the paragraph.

Tabs

Opens the Tabs dialog box to set tab stops.

Bullets

Starts each paragraph with bullets or numbers.

Alignment - Left, Right, Center

Sets the text left-aligned, right-aligned, or centered-aligned within the text region.
