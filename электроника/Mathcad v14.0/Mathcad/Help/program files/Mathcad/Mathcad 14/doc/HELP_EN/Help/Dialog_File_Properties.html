File Properties Dialog Box

The File Properties dialog allows you to view and edit some of the information associated with a worksheet, such as metadata and XML output options.

Summary tab

The Summary tab displays the file information associated with your worksheet. On it, you can set the following:

Title

The title associated with the worksheet. When you save the worksheet as either XML or HTML, Mathcad uses this value for the <title> tag.

Author

The person who created the worksheet. Defaults to the name supplied during installation.

Description

A brief summary of the worksheet, useful for noting the intended purpose of the document. When math regions are copied and pasted from the current worksheet into another worksheet, the description is displayed with other provenance information.

Company

The company associated with the worksheet. Defaults to the company name supplied during installation.

Keywords

A brief list of key terms. When you save the worksheet as an XML file you can search for these terms.

Automatically generated metadata

Other information is automatically controlled by Mathcad and displayed in the dialog:

  • Name - the filename used to save the worksheet. This is the same name displayed in the application (or document) titlebar.

  • Type - the current file format.

  • Location - the file path to the worksheet.

  • Size - the file size.

  • Created - the date and time the file was first saved.

  • Modified - the date and time the file was last saved.

  • Modified by - the last person to save the file.

  • Revision - the version number of the worksheet, updated every time the worksheet is saved.

Custom tab

The Custom tab allows you to specify custom document metadata. Enter a descriptive name for your metadata or choose one of the default names, specify its type, enter an appropriate value, and then click Add to save your entry. Saved metadata is displayed in a scrollable list located on the lower half of the dialog.

To edit existing metadata, select the entry you want to edit in the list, make your changes, and click the Modify button. To delete an entry, select it in the list, and click the Delete button.

XML Options tab

The options on the XML Options tab only affects worksheets saved as XML documents. The options on this tab are:

Region Image Rendering Options

You can choose to save math regions, graphs, and text as images in addition to XML markup. These images can be viewed in applications other than Mathcad (such as in a Web browser). PNG and JPEG image formats are available. If you select the JPEG format, you can also control the image quality. If you reduce the image quality, you can reduce the size of a file with many images in it.

Results Storage

Mathcad stores evaluated results, those numeric values displayed after an equals sign, when you save your worksheet. This allows you to view the last set of calculated results when you open the worksheet with Automatic Calculation disabled.

Some results — large matrices in particular — may dramatically increase the file size of your worksheet. To keep file sizes more manageable, Mathcad allows you to exclude large results from your saved worksheets. To do so, enable the "Don't store large evaluated results" option. Deselect this option to store all calculated results in the saved worksheet.
