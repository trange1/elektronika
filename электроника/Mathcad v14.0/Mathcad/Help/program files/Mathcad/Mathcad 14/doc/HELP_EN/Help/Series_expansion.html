Series Expansion of an Expression

To expand a function in a Taylor or Laurent series, select the function and either:

  • Press [Ctrl] [Shift] [.] and type the keyword "series."

    - or -

  • Click series on the Symbolic toolbar.

Then press [Enter].

If the function contains more than one variable, type a comma after "series," and then type a comma-separated list of the variable(s) around which you want to expand in the placeholder.

By default, Mathcad expands the function about the point 0, and returns the first six terms of the series. You can change the default behavior by including arguments after "series."

Notes:

  • Another method is to select just the independent variable for the function (not the entire function) and click Variable > Expand to Series ... on the Symbolics menu. In this case the result does not automatically update if you make changes elsewhere in the worksheet.
  • Taylor series for a function may converge only in some small interval around the center. Furthermore, functions like sin or exp have series with infinitely many terms (how many depends on the order you select). When you approximate a function by the polynomial returned by Expand to Series, the approximation is reasonably accurate close to the center, but may be quite inaccurate for values far from the center.
