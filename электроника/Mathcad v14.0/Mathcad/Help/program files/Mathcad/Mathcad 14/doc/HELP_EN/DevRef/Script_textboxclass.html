Mathsoft Control TextBox Classes

The Mathsoft Control TextBox object is a class that can be scripted using VBScript or JScript. The TextBox class encompasses the following properties. It has no methods or events associated with it.

Properties

Formatting Properties

  • AutoSize
  • BackColor and ForeColor
  • Border and ClientEdge
  • HorizontalScroll, LeftScrollBar and VerticalScroll
  • Lowercase and Uppercase
  • Multiline
  • NoHideSelection

State Properties

  • AutoHScroll and AutoVScroll
  • Enabled
  • HelpContextID
  • hWind
  • Number
  • OEMConvert
  • Password
  • ReadOnly
  • RightToLeftReadingOrder
  • Text
  • WantsReturn
