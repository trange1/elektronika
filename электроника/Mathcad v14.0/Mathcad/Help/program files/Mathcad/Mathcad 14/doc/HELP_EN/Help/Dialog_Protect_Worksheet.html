Protect Worksheet Dialog Box

The Protect Worksheet dialog box allows you to enable one of the three levels of protection for your worksheet. You can specify the level of protection to meet your needs. Worksheets can be password-protected, to prevent unauthorized users from disabling or changing worksheet protection.

File

Restricts the options in the Save As dialog to file types that cannot be opened and easily edited outside of Mathcad, then re-opened in Mathcad.

Attempts to save an XMCD worksheet with protection enabled result in an error. When you enable protection for an XMCD worksheet, you must use the Save As dialog to save the worksheet in the XMCDZ (or other preferred) format.

Content

Includes file protection. Additionally:

  • Existing regions are protected by default and cannot be changed. Controls are frozen in their current state when you enable protection.
  • New regions can be created.
  • Protected regions can be copied.

Editing

Includes content protection. Additionally:

  • Protected regions cannot be copied.
  • No new regions can be created in the worksheet.

Password

Prevents unauthorized users from disabling or changing the protection level assigned to the worksheet.
