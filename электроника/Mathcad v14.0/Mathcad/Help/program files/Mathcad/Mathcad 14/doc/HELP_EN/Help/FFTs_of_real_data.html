FFT(v) Returns the Fast Fourier Transform of a vector v. The formula is equivalent to fft, but is scaled by 1/n instead of 1/√n, and uses a negative exponent going from the time to the frequency domain.

IFFT(u) Returns the inverse Fourier transform for u created with FFT. The formula is equivalent to ifft, but is scaled by 1 instead of 1/√n, and uses a positive exponent going from the frequency to the time domain.

Arguments:

  • v is a real-valued vector with 2m elements (m > 2), representing samples at regular time intervals.
  • u is a complex-valued vector with 1 + 2m-1 elements (m > 2), representing samples at frequencies.

To save time and memory, Mathcad does not calculate the second half of the frequency spectrum. For vectors with complex values, or with an arbitrary number of elements, use cfft or CFFT instead. Likewise, for 2-dimensional Fourier transforms, use cfft/CFFT instead.
