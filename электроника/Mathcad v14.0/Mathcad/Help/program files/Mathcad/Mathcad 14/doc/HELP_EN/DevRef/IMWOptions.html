Options Property

The Options property gets or sets options in a Worksheet object.

Syntax

variant = objMC.Options(MCWorksheetOption WorksheetOption)

objMC.Options(MCWorksheetOption WorksheetOption) = variant

Element

Description

variant

The worksheet option to set or return. The content of variant varies with the option type.

objMC

A Mathcad Worksheet object.

MCWorksheetOption

The worksheet option to send to or return from Mathcad.
