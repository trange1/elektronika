MCFileFormat Enumeration

Supported Formats

The MCFileFormat enumeration specifies the file format in the various places where a worksheet can be saved.

Name

Value

Description

mcDefaultFormat

0

The format of the worksheet when opened.

mcPromptUser

1

Allows the user to pick from a list of available formats.

mcHTML

3

Static HyperText Markup Language page.

mcMcadCurrentVersion

9

Current Mathcad version format.

mcMcad12

12

Mathcad 12 worksheet.

mcXMCD12

13

Mathcad 12 XML worksheet.

mcXMCDZ12

14

Mathcad 12 compressed XML worksheet.

mcXMCT

15

Mathcad XML Template.

mcRTF

16

Mathcad RTF worksheet.

mcMcad11

17

Mathcad 11 worksheet.

mcXMCD13

18

Mathcad 13 XML worksheet.

mcXMCDZ13

19

Mathcad 13 compressed XML worksheet.

mcXMCD14

20

Mathcad 14 XML worksheet.

mcXMCDZ14

21

Mathcad 14 compressed XML worksheet.

Note:

  • The value of mcMcadCurrentVersion depends on the format of the individual worksheet. XMCD worksheets save as XMCD when using mcMcadCurrentVersion, while XMCDZ worksheets save as XMCDZ.
