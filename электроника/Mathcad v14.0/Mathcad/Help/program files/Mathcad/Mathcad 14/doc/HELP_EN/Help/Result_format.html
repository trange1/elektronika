Working with Numerical and Symbolic Results

To change the default display of numerical and symbolic results in a worksheet:

  1. Click in a blank area of the worksheet.

  2. Choose Result from the Format menu to see the Result Format dialog box, and choose your default settings.

  3. Make sure that the button labeled "Set as default" is checked, and click "OK."

To change the display of a particular result, click on the equation, and follow the same steps.

To affect symbolic results, be sure to check the box in the lower left-hand corner, "Apply to symbolic results."

Viewing a Numerical Result in Full Precision

All arithmetic is carried out with full precision regardless of how the number appears in your worksheet. You can display a particular result in full precision permanently by following the steps above.

Displayed precision for a result can be set up to 17 decimal places. You may still see some truncation or rounding of values if your zero and complex tolerances are set too large. Click on the Tolerance tab of the Result Format dialog box to adjust these settings.

Notes:

  • You can view the full precision of a single value but not an array of values. To see the full precision of a resulting array, change its format settings.

  • If you wish to truncate the number that is used in subsequent calculations to whatever precision you've chosen for display, use the round function to choose the appropriate number of places. You may also wish to adjust the tolerance for your calculations.

  • Common defaults changes can be saved to a template, so new documents have your preferred settings.

Copying Numerical Results

Any calculation results in Mathcad can be copied and pasted, either into Mathcad or into other applications.

Numerical results are copied only to the displayed precision. To copy a result in greater precision, double-click it and increase "Displayed Precision" on the Result Format dialog box.
