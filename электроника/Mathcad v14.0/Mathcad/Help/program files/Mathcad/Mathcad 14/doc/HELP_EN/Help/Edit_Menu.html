Edit Menu Commands

Undo

Reverses the last action(s) performed. The default undo history is 100 actions, which can be changed in the Preferences dialog.

Redo

Reverses the most recent undo action(s).

Cut

Copies the selection and deletes it so that you can paste it elsewhere.

Copy

Copies the selection so that you can paste it elsewhere.

Paste

Pastes content into the worksheet.

Paste Special

Pastes content in a specified format.

Delete

Deletes selection but does not store it for pasting. You can reverse a delete using the Undo command.

Select All

If the cursor is in a text region, selects all text in that region. Otherwise, selects all the regions in a worksheet.

Find

Displays the Find dialog, allowing you to search the worksheet for letters, numbers, or special characters in math and/or text regions.

Displays a dialog allowing Designate users to find text and values across worksheets in their repository.

Replace

Displays the Replace dialog, allowing you to substitute one set of characters in math and/or text regions for another.

Go to Page

Goes to top of the selected page in the worksheet.

Links

Edits OLE links in the worksheet. OLE links are created by either using Paste Special to paste a link to a file or inserting a link to an OLE object.

Object

Edits or opens an OLE object in the worksheet. Select the object to make this menu item available.
