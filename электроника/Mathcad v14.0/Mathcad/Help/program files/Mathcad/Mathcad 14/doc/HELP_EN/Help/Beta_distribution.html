Beta Distribution

Distribution generators for the Beta Equation:

dbeta(x, s1, s2) Returns the probability density for value x.

pbeta(x, s1, s2) Returns the cumulative probability distribution for value x.

qbeta(p, s1, s2) Returns the inverse cumulative probability distribution for probability p.

rbeta(m, s1, s2) Returns a vector of m random numbers having the beta distribution.

Arguments:

  • x is a scalar or vector of real values between 0 and 1. To allow integration and other operations over this argument, values outside of the stated range are allowed, but they produce a 0 result.
  • s1 and s2 are real shape parameters, greater than 0.
  • p is a real probability between 0 and 1.
  • m is an integer greater than 0.

Note:

  • For definitions of the functions, and information on generating random numbers, see the Density and Distribution definitions.
