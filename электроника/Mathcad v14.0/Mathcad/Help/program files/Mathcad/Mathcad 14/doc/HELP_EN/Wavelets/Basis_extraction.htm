Basis extraction

extract_basis(T,b)

Returns coefficients in the basis specified by b from a 1D or 2D packet table T.

Arguments:

  • T is a vector of nested arrays representing a 1D or 2D wavelet packet table or a 1D or 2D cosine packet table.
  • b is a 2- or 3-column matrix representing a packet basis.
