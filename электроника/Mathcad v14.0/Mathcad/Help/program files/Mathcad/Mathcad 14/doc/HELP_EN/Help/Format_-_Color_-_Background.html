Format > Color

Background

Allows you to change the background color of the main worksheet window.

Highlight

Allows you to change the color used when you highlight a region.

Annotation

Allows you to set the color used for annotation parentheses around math regions with annotations and audit trail data. The same color is used to highlight any changes you've made in an E-book or the Resources Window.

Use Default Palette

Uses a single default 256-color palette for display if your display only supports 256 colors. If not checked, the worksheet uses whatever color palette is best for each picture in the worksheet. This may cause the screen to flash as you scroll through the worksheet, as each 256-color bitmap loads a new palette.

Optimize Palette

Optimizes the default color palette to the best possible 256-color palette to display all bitmaps in the worksheet for displays only supporting 256 colors.
