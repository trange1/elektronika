The Data Analysis Extension Pack contains a collection of new built-in functions and a new data importing component to help you acquire, analyze, and manage data in Mathcad. The functionality in this Extension Pack includes:

  • Matrix and data utility functions
  • Statistics
  • Data preconditioning
  • Principal Component Analysis
  • Interpolation
  • Regression

Note that there are many functions for data analysis already built into Mathcad. These are documented in the main Mathcad Help. They are also given greater elaboration and demonstration by examples in the E-book which accompanies this Extension Pack. We recommend that you use the E-book for a complete demonstration of available functions.
