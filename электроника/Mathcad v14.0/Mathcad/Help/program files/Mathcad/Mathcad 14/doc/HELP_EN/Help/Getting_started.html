Getting Started with Mathcad

When you start Mathcad, you see a blank worksheet in which you can enter text, equations, graphs, and images.

We recommend that new users start with the online Tutorials. The Getting Started Primers guide you through the fundamentals of Mathcad, while Features In-Depth provide more advanced lessons.
