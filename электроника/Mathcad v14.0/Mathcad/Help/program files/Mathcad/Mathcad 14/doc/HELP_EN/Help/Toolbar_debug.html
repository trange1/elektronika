Debugging Toolbar

The Debug toolbar is used with debugging functions to report on intermediate values in programs. To bring up the Debug toolbar, choose View > Toolbars > Debug.

The buttons on the Debug toolbar allow you to

  • Resume a paused calculation
  • Interrupt and exit a paused calculation (the same as pressing Esc)
  • Toggle debugging mode on and off
  • View or hide the Trace Window

To learn what each button does, hover your cursor over the button until a tooltip shows the title and a description appears on the message line of the Status Bar.

These buttons are shortcuts to the commands in the Tools > Debug menu as well as the View > Trace Window menu item.
