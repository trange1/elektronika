Complex Conjugate

Keystroke: ["]

Returns the conjugate of x: Re(x) − i·Im(x), where i is the imaginary unit.

Operands:

  • x is a scalar, which is selected when the operator is applied.

Notes:

  • This keystroke only produces the complex conjugate operator when an expression is selected. If used in an empty placeholder, ["] produces a string variable. If used in a blank region of the worksheet, ["] produces a text region.
