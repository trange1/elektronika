Updating Mathcad

New versions of the Mathcad application periodically become available for download from the PTC Web site. To check if you have the most recent release of your version of Mathcad, choose Mathcad Update from the Help menu. Mathcad automatically determines if an update is available. If an update is available, your default browser launches and directs you to the Mathcad Download area of the PTC Web site.

Notes:

  • In order to use this feature you must be connected to the Internet.

  • No information from your computer is sent to the PTC Web site during the update process.
