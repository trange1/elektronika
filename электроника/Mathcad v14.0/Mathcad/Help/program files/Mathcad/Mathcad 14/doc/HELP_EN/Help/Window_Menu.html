Window Menu

Cascade

Hides inactive worksheet windows behind the active window without obscuring any title bars.

Tile Horizontal

Arranges all worksheet windows one above the other, with the active worksheet on top.

Tile Vertical

Arranges all worksheet windows side-by-side, with the active worksheet on the left.

Open Worksheets List

Lists open worksheets. The one with the check is the active worksheet.

More Windows

Brings up a list of all open worksheets when you have more than nine open worksheets.
