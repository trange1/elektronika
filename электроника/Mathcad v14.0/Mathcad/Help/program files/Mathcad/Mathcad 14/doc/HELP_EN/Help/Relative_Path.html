Relative Paths to Files

Use a relative path to store the location of a linked file relative to the source Mathcad worksheet. A relative path keeps the link even if you move the linked file and the main worksheet, but keep the relative directory structure between the two the same.

If you specify a relative path, the path may display as a full path but Mathcad uses a relative path internally. If you move the files, the displayed path changes accordingly.

Notes:

  • You must move the files through Windows Explorer, not by copying them or using "Save As" from within Mathcad. Otherwise, the relative paths are broken.

  • A capital "R" appears in parentheses at the end of the path for the reference in your worksheet.
