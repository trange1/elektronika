Wavelet transform filtering

iwave2d(M, n) Returns the n-level inverse wavelet transform of M.

wave2d(M, n) Returns the n-level wavelet transform of M.

wavescale(M, n, Min, Max) The n-level wavelet transform of M, scaled between Min and Max.

Arguments:

  • M is an image matrix.
  • n is a positive integer, or 0.
  • Min, Max are real numbers.
