Finding Characters

Allows you to search for characters in both text and in variable and function names. However, certain characters can only be located inside text or strings.

Find Dialog Box

The Find dialog box has the following options:

Find what

Enter the characters to find.

Match whole word only

Locates a whole word rather than part of a word.

Match case

Locates characters having the same case as those you enter (uppercase or lowercase).

Find in Text Regions

Locates characters in text regions only.

Find in Math Regions

Locates characters in math regions only.

Find in Collapsed Areas

Also locates characters in collapsed areas.

Up

Searches above the location of the crosshair.

Down

Searches below the location of the crosshair.

Find Next

Locates the next or previous occurrence of the characters.
