Defining Headers and Footers

To add headers and footers to your printout:

  • Choose Headers and Footers from the View menu to open the Header and Footer dialog box.

  • Click on either the "Header" or the "Footer" tab.

  • Type your text in either the left, right, or center text box.

  • Use the buttons in the Tools section of the dialog to insert information such as page numbers, the date, or the time.

  • See the Header/Footer dialog box for other options.

  • To see exactly how the header or footer will look, choose Print Preview from the File menu.
