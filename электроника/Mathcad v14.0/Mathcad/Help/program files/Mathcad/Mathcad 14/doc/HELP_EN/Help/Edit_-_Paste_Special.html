Edit - Paste Special

Allows you to paste content in a specified format. For example, to paste a picture taken from an OLE server as a bitmap rather than embedded or linked object, choose Paste Special and select "bitmap" from the Paste Special dialog box.

Paste Special Dialog Box

The Paste Special dialog box has the following options:

Paste

Embeds the copied information into the worksheet. Changes you make to an embedded object affect only the copy in the worksheet. The original object is unchanged.

Paste Link

Links the copied information into the worksheet. When you make changes to a linked object, you change both the copy in your worksheet as well as the original.

This option is available only when an object was copied from a saved file.

Paste As

Specifies how the copied information is pasted into the worksheet. The available options depend on the type of information you copied.

Display As Icon

Displays the pasted information as an icon. Double-click on the icon to open the object.
