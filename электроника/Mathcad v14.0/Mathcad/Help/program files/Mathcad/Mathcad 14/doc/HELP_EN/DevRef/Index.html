Mathcad has a broad range of interfaces for publishing, exporting and importing data, and software. The Developer's Reference compares and documents these interfaces, so you can customize Mathcad for your own use. Sections of this reference assume that you are familiar with associated technologies, such as scripting languages, XML, HTML, C or C++, and development environments, such as Visual Studio. Sample applications and code snippets are provided for explanation.
Mathcad's interfaces allow you to:
  • Incorporate custom or legacy algorithms in C, C++, or FORTRAN.
  • Control variables in a Mathcad document with forms controls.
  • Access Windows applications that have Automation interfaces.
  • Access Mathcad's Automation interface from external applications that support scripting.
  • Create new Mathcad components for external applications using Windows' COM interface.
  • Create a hyperlinked, searchable electronic book, or create static web sites with templates. You can also create dynamic web sites with the Mathcad Calculation Server. These options are discussed in the Author's Reference.
  • Export Mathcad documents to other XML or text applications, with XSLT.
