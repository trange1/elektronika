Entering a Table of Numbers

As an alternative to the Insert Matrix command, you can create an array by choosing Data from the Insert menu, then choosing Table from the resulting list. A table with row and column labels appears in your worksheet, with a blank placeholder to the left of the := operator. In the placeholder, enter the name of the array. You can enter numbers directly into table cells by clicking on them.

Notes:

  • You cannot create a data table in an existing expression, you must begin from a blank region of the worksheet.
  • Numbers can be entered in decimal or engineering format, for example, 3.4, 2i, or 4e-6. Enter only numbers into a table, not mathematical expressions.
  • Each row must have the same number of data values. If you do not enter a number into a cell, 0 is inserted into the cell.
  • You can paste data into a data table, or import a file. Most data types are correctly retained on import or paste: strings and dates paste as Mathcad string variables, surrounded by quotes.
