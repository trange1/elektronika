Program Debugging

trace(S, x, y, z, ...)
pause(S, x, y, z, ...)

If debugging mode is on, these functions print a string to the Trace Window. The pause function pauses execution of the worksheet when Debug mode is on. Execution can be resumed or interrupted using buttons on the Debug toolbar. The printed string contains the value of the arguments x, y, z, ... with print order and surrounding text specified by S. If the argument is an n-element vector, v, the result has the format

"vector{n} [v0, v1, ..., vn]"

where v0, v1, ..., vn are the values of each entry. If the argument is an m × n matrix, M, the result has the format

"matrix{m, n} [[M0,0, M0,1, ..., M0,n], ..., [Mm,0, Mm,1, ..., Mm,n]]"

Both functions return 0 to the worksheet.

Arguments:

  • S (optional for one-variable case) is a string containing control codes to specify location of traced argument(s) in the output. Each argument is referenced as {#}, starting with 0. For example,

    x := 0.2·sec     y := 30·ms

    trace("x = {0}, y = {1}", x, y) = "x = 0.2, y = 0.03"

  • x, y, z, ... are any valid Mathcad expressions.

Notes:

  • Debugging mode must be on and the Trace Window must be visible to see the traced output and to pause calculations.
  • Values with units are scaled to the base SI units. To scale a united value specifically, divide by the desired unit in the argument to the trace or pause function.
  • Pausing and then interrupting a calculation is equivalent to pressing [Esc].
