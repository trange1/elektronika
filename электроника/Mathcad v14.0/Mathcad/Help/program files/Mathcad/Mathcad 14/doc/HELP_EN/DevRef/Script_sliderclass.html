Mathsoft Control Slider Class

The Mathsoft Control Slider object is a class that can be scripted using VBScript or JScript. Some formatting, such as font changes, is controlled from the Properties dialog only. The Slider class encompasses the following properties, methods, and events:

Properties

Formatting Properties

  • AutoTicks, TickFrequency and Ticks
  • Border, ClientEdge and StaticEdge
  • NumTicks
  • Orientation
  • Point
  • Transparent

State Properties

  • EnableSelection
  • LineSize and PageSize
  • MaxRange and MinRange
  • Position
  • SelectionLow and SelectionHigh

Methods

  • Recalculate

Events

  • ValueChanged
