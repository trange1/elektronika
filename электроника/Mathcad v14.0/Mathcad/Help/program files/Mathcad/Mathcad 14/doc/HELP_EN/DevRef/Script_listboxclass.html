Mathsoft Control ListBox Class

The Mathsoft Control ListBox object is a class that can be scripted using VBScript or JScript. Some formatting, such as font changes, is controlled from the Properties dialog only. The ListBox class encompasses the following properties, methods, and events:

Properties

Formatting Properties

  • Border and ClientEdge
  • DisableNoScroll
  • HorizontalScroll and VerticalScroll
  • ModalFrame and StaticEdge
  • NoIntegralHeight
  • RightAlignedText
  • Sort
  • Transparent

State Properties

  • Count
  • CurSel
  • LBERR and LBERRSPACE
  • RightToLeftReadingOrder

Methods

  • AddString, DeleteString and InsertString
  • FindString and FindStringExact
  • GetText
  • Recalculate
  • ResetContent
  • SelectString

Events

  • DblClick
  • Error
  • SelectionChange
