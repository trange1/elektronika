Format > Graph

Controls the default characteristics of plots in the worksheet. If a plot is selected when this menu option is chosen, you change characteristics for that plot only. If nothing is selected, you can change the 2D plot defaults for the entire worksheet.

X-Y plots and Polar Plots

By selecting Format > Graph > XY Plot or Polar Plot, you can control tick marks, gridlines, graph types, line types, symbol types and other attributes through the Axes and Traces pages. If a particular 2D plot is selected, you can specify labels and defaults for the plot as well. Double-clicking on a plot also brings up these dialogs, as does right-clicking on the plot and choosing Format.

3D Plots

Select Format > Graph > 3D Plot to open the 3D Plot Format Dialog.

Formatting for 3D plots is only active when a 3D plot is selected; it cannot be set for the document as a whole. Double-clicking on a plot also brings up these dialogs, as does right-clicking on the plot and choosing Format.

You can also use the General tab on the Format dialog to change from one plot type to another. Not all formatting options are available for all plot types. Click on the 3D plot types below to learn more about the settings you may want to control:

  • Surface

  • Contour

  • 3D Scatter

  • 3D bar plot

  • Vector field

Trace for 2D Plots

Allows you to read coordinates directly from the selected plot. The same dialog is accessible by right-clicking on the plot and choosing Trace. This feature is available for 2D plots only.

Zoom

Allows you to magnify the selected plot. The same dialog is accessible by right-clicking on the plot and choosing Zoom. This dialog is for 2D plots only. To zoom a 3D plot, use the mouse or the Format dialog.
