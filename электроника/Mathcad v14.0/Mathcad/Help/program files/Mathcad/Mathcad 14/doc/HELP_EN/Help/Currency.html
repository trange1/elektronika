Using Currency in Mathcad

Mathcad includes support for currency in your calculations. You can assign a currency unit to variables, which allows you to pass monetary values to functions and format your results accordingly.

All unit systems include support for the Unicode base currency unit, ¤, with a value of 1. To change this to another currency:

  1. Click Tools > Worksheet Options... to open the Worksheet Options dialog.
  2. Select the proper Currency Symbol on the Display tab.

To use the currency unit in your calculations:

  1. Select Money from the Insert Unit dialog to apply the currency unit to a variable, or
  2. Use the Worksheet Options dialog to set the currency symbol, then type the symbol in your worksheet.

Notes:

  • Unlike other types of units, Mathcad does not maintain conversion rates between different currencies, and thus cannot scale between them. Money is quantified with a single unit, which is displayed as a specific currency according to user preference. You can, however, define and scale other currencies relative to the base currency within your worksheet.
  • Some currency symbols, such as $, €, and ¥ cannot be entered directly on some keyboards. For instructions on how to use these special characters, see the Help topic "Typing Reserved Keystrokes and Special Symbols in a Name."
  • Many currency symbols are available in the new Mathcad font Mathcad UniMath, which is distributed with Mathcad. See the Mathcad QuickSheet, Extra Mathcad Symbols, for currency symbols, and instructions on how to find other symbols to insert into your worksheets.
