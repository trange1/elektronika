Text Styles

Text styles give you an easy way to create a consistent appearance in your worksheets. Using text styles enables you to quickly apply text formatting to your text regions.

Available text styles depend on the template that was used to create your worksheet. You can examine the different templates and text styles provided in Mathcad by choosing New from the File menu, and modify them or create new ones for your own use. The default worksheet uses the NORMAL.XMCT template.
