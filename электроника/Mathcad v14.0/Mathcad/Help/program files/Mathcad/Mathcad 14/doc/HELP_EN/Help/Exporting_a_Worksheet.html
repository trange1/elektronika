Exporting to Microsoft Word

To export a worksheet to Microsoft Word, you can save the worksheet as an RTF file, or you can copy and paste the worksheet regions.

Exporting as RTF

You can turn an entire worksheet, complete with equations and plots, into an RTF file readable by Microsoft Word. RTF can represent not only simple text but also graphics, equations, fonts, and document layout information like page breaks, margins and paragraph formats. The layout matches the original worksheet and the text can be further edited in Word. All math regions and graphs become images in the Word document.

Since the default margins in Word are different from those in Mathcad, we recommend using the Microsoft Word template under New in the File menu. Or you can set your margins in the Mathcad worksheet to match the margins in Word, using Page Setup under the File menu in both programs.

Note that anything beyond the right margin in a Mathcad worksheet will not appear in the Microsoft Word document.

To save a Mathcad worksheet as RTF for Microsoft Word:

  1. Use the Microsoft Word template in Mathcad or set your margins to match those in the Word document.

  2. Make sure the material you want to export is within the right margin.

  3. Choose Save As from the File menu.

  4. Choose Rich Text Format (*.rtf) from the "Save as Type" drop-down list in the Save As dialog box.

  5. In the File Name box, type the name of the RTF file you want to create.

When you open this RTF file with Microsoft Word, you may find all the regions lined up one above the other at the left edge of the document. If so, switch to Word's Print Layout under the View menu to more accurately display your worksheet's appearance.

Copying and Pasting

Another way to export a worksheet is to copy one or multiple regions in the worksheet and paste them into another application. To do so:

  1. Choose Select All from the Edit menu to copy all regions or select individual regions by dragging over them, or [Shift]-clicking on each one.

  2. Choose Copy from the Edit menu.

  3. Open Microsoft Word and paste your selection.

Under Paste Special, you have several options. If you choose Mathcad Document Object, you can edit math regions with Mathcad's toolbars and menus, which open inside Word. You can also choose to import the Mathcad regions as pictures, unformatted text, or RTF (formatted text).
