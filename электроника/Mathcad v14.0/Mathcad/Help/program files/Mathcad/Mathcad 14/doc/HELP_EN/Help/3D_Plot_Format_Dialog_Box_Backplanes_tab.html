3D Plot Format Dialog Box (Backplanes Tab)

To format the X-Y, Y-Z, or X-Z backplane, click on the appropriate tab inside the dialog box.

Fill Backplane, Color

Fills the backplane with a color specified in the box next to Color.

Backplane Border

Draws a border around the backplane.

Draw Lines

Draws grid lines or sub-grid lines on a particular axis.

Note: The grid lines are controlled by the options on the Axes tab as well. If you set the grid lines on the Backplanes tab, you set them for that backplane only.

Draw Ticks

Draws tick marks at the grids or sub-grids on a particular axis.

Note: The tick marks are controlled by the options on the Axes tab as well. If you set the tick marks on the Backplanes tab, you set them for that backplane only.

Line Color

Specifies the color of the grid lines or sub-grid lines.

Line Weight

Specifies the width of the grid lines or sub-grid lines.
