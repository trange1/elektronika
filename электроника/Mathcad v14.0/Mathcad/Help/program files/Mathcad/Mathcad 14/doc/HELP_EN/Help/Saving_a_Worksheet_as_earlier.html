Saving a Worksheet as an Earlier Version

To save a worksheet so you can open it with an earlier version:

  1. Choose Save As from the File menu.

  2. In the "Save as type" drop-down list, select the version you'd like the file to be and provide a file name.

  3. Click "Save." A message may appear warning you that certain features do not work in an earlier version.

When you open a worksheet saved as an earlier version, you may see an additional warning indicating that certain features are not supported. Features that are only available in the version the worksheet was created in appear as bitmaps in an earlier version and any regions that depend on them issue error messages.
