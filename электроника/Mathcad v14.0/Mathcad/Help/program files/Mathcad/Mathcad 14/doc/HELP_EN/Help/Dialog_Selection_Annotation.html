Sources Dialog

The Sources dialog allows you to note and track the sourcing of expressions in a Mathcad worksheet.

Comment tab

The Comment tab allows you to make comments for all or part of an expression in your worksheet. As you copy and paste the expression into another worksheet, any comments are copied along with it.

To comment an expression, just right-click on it, then enter your comments into the Selection Annotation dialog.

When annotating a subexpression, such as 1/2·b·h, not all of the selected combinations can be annotated. If the Selection Annotation dialog is unavailable, place parentheses to group the subexpression, such as 1/2·(b·h), and then make your comments.

To see expressions with comments:

  • Choose View > Annotations to see green parentheses around the expressions that have been annotated, or
  • Right-click on an expression and choose View/Edit Annotation.

View Sources Dialog

View Sources is visible for an expression once you have copied it to a new worksheet. To see expressions with sources:

  • Choose View > Annotations to see green parentheses around the expressions that have been annotated, or
  • Right-click on an expression and choose View Sources.

On the Sources tab, you can see the original source for the expression and information about the worksheet from which it was most recently copied.
