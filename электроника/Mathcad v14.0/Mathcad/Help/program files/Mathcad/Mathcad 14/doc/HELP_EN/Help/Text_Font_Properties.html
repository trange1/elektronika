Changing Text Font Properties

To modify the font, style, size, color, or position of text:

  1. Select the text.

  2. Choose Text from the Format menu.

  3. Choose from among the various font properties available in the dialog box.

Many of the options in the Text Format dialog are also available via the buttons and drop-down lists on the Formatting toolbar.

Note that in order to change all the text based on a particular text style, you need to modify the style properties or apply a different style.
