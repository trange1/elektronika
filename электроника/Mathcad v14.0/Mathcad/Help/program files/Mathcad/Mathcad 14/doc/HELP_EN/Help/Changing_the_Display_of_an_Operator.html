Changing the Display of an Operator

When you type an operator into a worksheet, it has a certain default appearance. There are some operators whose appearance you can control. These include:

Definition symbol

Global definition symbol

Symbolic evaluation symbol

Multiplication

Derivative

Literal subscript

Boolean Equal to

To change the way any of these operators is displayed throughout a worksheet:

  1. Choose Worksheet Options from the Tools menu.

  2. Click the Display tab and use the drop-down options next to each operator to select a display option.

To change the appearance of an operator in individual expressions:

  1. Right-click on the operator.

  2. Choose View [operator name] As from the menu.
