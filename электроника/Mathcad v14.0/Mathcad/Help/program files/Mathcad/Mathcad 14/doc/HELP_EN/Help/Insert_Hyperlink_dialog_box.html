Insert Hyperlink Dialog Box

Link to file or URL

Enter the path or browse to a local or remote file. For example, enter a path such as:

c:\windows\file.xmcd

- or -

http://www.ptc.com/go/mathsoft/text.xmcd

Use relative path for hyperlink

Stores the location of the linked file relative to the worksheet containing the link. Maintains the link if you move the two linked files as long as you keep the relative directory structure between them the same. If you save the file using Save As, the link path remains the same, so be sure to copy the linked file to the new directory.

For relative links, the two worksheets must be on the same local or network drive.

The relative path may display as a full path, but the relative path is saved internally.

Display as pop-up document

Opens the linked worksheet in its own window.

Message that appears on status line when mouse is over link

Displays the text you type in.

Remove Link

Deactivates the hyperlink associated with the selection.
