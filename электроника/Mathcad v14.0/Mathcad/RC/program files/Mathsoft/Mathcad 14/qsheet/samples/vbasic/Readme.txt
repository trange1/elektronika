NOTE:
====
The library MSVBVM50.DLL must be present in your
C:\WINDOWS\SYSTEM\ directory in order for you to be
able to run the Visual Basic projects located in
this folder.

To run a VB application compiled under Visual Studio .NET, you must be using Windows XP, or you must download and install a .NET Framework from Microsoft at http://www.microsoft.com. 

If you choose to move the executable file in the VB Automation Example, you must place the Interop.Mathcad.dll file in the same directory as the Mathcad Automation Example.exe or put it in the Global Access Cache (GAC). For more information on the GAC, please visit http://msdn.microsoft.com/library/default.asp?url=/library/en-us/cpguide/html/cpconinstallingassemblyintoglobalassemblycache.asp
