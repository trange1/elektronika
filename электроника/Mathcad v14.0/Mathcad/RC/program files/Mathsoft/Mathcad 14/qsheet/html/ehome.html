My Site

With Mathcad 14 you can specify your preferred home page for use within the Resources window.

Set your preferred Web site's address as the My Site URL on the File Locations tab of the Preferences dialog (available from the Tools menu) to enable direct access to the site from within your Mathcad environment.

Use the Resources links to browse to Resource window files. Use the Quick Launch links for access to current information posted on PTC's product and technical support websites.
