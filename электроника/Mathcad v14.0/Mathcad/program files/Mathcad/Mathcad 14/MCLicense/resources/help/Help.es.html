Página de bienvenida

El asistente para la instalación de la licencia de Mathcad permite registrar la instalación de Mathcad. Si fuera necesario, le guiará durante el proceso para solicitar el fichero de licencia.

Los usuarios de Enterprise Edition deben especificar el nombre y, si fuera necesario, el número de puerto de sus servidores con licencia cuando se les solicite.

A los usuarios de Single User Edition se les dirige al sitio de gestión de licencias de PTC. Inicie sesión en su cuenta de usuario de PTC y solicite su fichero de licencia. Si no tiene una cuenta de usuario de PTC, puede crear una cuando solicite el fichero de licencia. El asistente guarda automáticamente el fichero de licencia y configura Mathcad para que lo busque al inicio. Si ya tiene un fichero de licencia válido, puede utilizar este asistente para que Mathcad lo utilice.

Una vez que Mathcad conoce la ubicación del fichero de licencia o del servidor utiliza esta información cada vez que se inicia. Si Mathcad no puede encontrar el fichero de licencia o el servidor, inicia este asistente. Si esto ocurre, vuelva a indicar a Mathcad la ubicación del fichero de licencia.

Mecanismo de configuración

Esta página es para usuarios de Single User Edition únicamente. Los usuarios de Enterprise Edition que lleguen a esta página, deben ponerse en contacto con su administrador de sistemas.

Indique si necesita obtener un fichero de licencia de PTC o configurar Mathcad para que utilice un fichero de licencia existente.

Para obtener un fichero de licencia de PTC, debe tener acceso a Internet en su estación de trabajo y una cuenta de usuario de PTC. Si no tiene una cuenta de usuario de PTC, puede crear una cuando solicite el fichero de licencia.

Acerca de las cuentas de usuario de PTC

Su cuenta de usuario de PTC proporciona acceso a las funciones de soporte del sitio Web de PTC (PTC.com), incluyendo gestión de licencias. Además, puede registrarse para eventos y retransmisiones vía Web, recibir descargas gratuitas y descubrir sugerencias, trucos y tutoriales para productos de PTC.

Código de producto de Mathcad

Esta página es para usuarios de Single User Edition únicamente. Los usuarios de Enterprise Edition que lleguen a esta página, deben ponerse en contacto con su administrador de sistemas.

Indique el código de producto que ha recibido al adquirir Mathcad. Si no ha recibido un código de producto porque tiene una copia de evaluación de Mathcad, seleccione la opción licencia de prueba.

Si ha seleccionado la opción licencia de prueba, el asistente configura Mathcad para que se inicie normalmente hasta que expire el periodo de evaluación. Cuando termina el periodo de evaluación, Mathcad le solicitará un fichero o servidor de licencia.

Si obtiene o intenta obtener una licencia de Mathcad antes de que expire el periodo de evaluación, inicie Mathcad desde una línea de comandos, seguido del texto " /license":

"C:\Archivos de programa\Mathcad\Mathcad 14\mathcad.exe" /license

El asistente para la instalación de la licencia de Mathcad inicia la aplicación y permite registrar la instalación de Mathcad.

Creación de una cuenta

Si ya dispone de cuenta de usuario de PTC, pulse Siguiente para introducir el Nombre de usuario y Contraseña. El asistente para la instalación de la licencia utiliza esta información para acceder al sitio Web de PTC y obtener automáticamente un fichero de licencia para Mathcad.

Si no dispone de una cuenta de usuario de PTC, pulse Crear cuenta (Create Account) para abrir una ventana del explorador al sitio Web de PTC, donde podrá crear una cuenta. Una vez haya creado la cuenta, vuelva al asistente para la instalación de la licencia y pulse Siguiente para solicitar un fichero de licencia, utilizando su cuenta.

Acerca de las cuentas de usuario de PTC

Es necesario tener una cuenta de usuario de PTC para obtener una licencia de Mathcad y acceder a otras funciones de soporte de productos del sitio Web de PTC (PTC.com).

Información de la cuenta

Introduzca su Nombre de usuario y Contraseña de PTC en los espacios correspondientes. El asistente utiliza esta información para acceder al sitio Web de PTC y solicitar automáticamente un fichero de licencia para Mathcad.

Pulse Finalizar para solicitar su fichero de licencia. El asistente guarda automáticamente el fichero de licencia en la carpeta de licencias de la instalación de Mathcad y configura la instalación para que utilice este fichero al iniciarse.

Si no dispone de una cuenta de usuario de PTC, pulse Volver para volver a la página Crear cuenta del asistente, donde podrá crear su propia cuenta.

Ubicación de la licencia

Usuarios de Single User Edition: Indique la ubicación del fichero de licencia o pulse ... para inspeccionar. Pulse Finalizar para configurar Mathcad.

Usuarios de Enterprise Edition: Especifique un fichero o un servidor de licencia para Mathcad. La mayoría de usuarios seleccionan Utilizar un servidor de licencia (Use a license server). Si selecciona un servidor de licencia:

  1. Indique el nombre del servidor de licencia.
  2. Indique el número de puerto que se utilizará para el servicio de licencia. El valor predefinido (7788) aparecerá automáticamente.

La comunicación entre el servidor y la estación de trabajo se produce en el puerto específico del servidor. Si no conoce el número de puerto correcto, póngase en contacto con su administrador de sistemas.

Si el administrador de sistemas le ha proporcionado un fichero de licencia para la estación de trabajo, seleccione Utilizar un fichero de licencia (Use a license file). Indique la ubicación del fichero de licencia o pulse ... para inspeccionar.

Una vez haya especificado el servidor de licencia o el fichero, pulse Finalizar para configurar Mathcad.

Instrucciones para obtener la licencia de forma manual

Si no dispone de acceso a Internet en el equipo que tiene Mathcad instalado, puede solicitar una licencia manualmente, en lugar de hacerlo mediante el asistente.

Para solicitar una licencia manualmente, póngase en contacto con el departamento de gestión de licencias de PTC. Existe una lista de números de teléfono disponible en el sitio Web de PTC en http://www.ptc.com/olm/contacts.htm.

Ya está listo

El servidor de licencia o la ubicación del fichero se ha registrado satisfactoriamente. Mathcad utilizará esta información la próxima vez que se inicie. Si Mathcad no puede encontrar el fichero de licencia o el servidor, inicia este asistente. Si esto ocurre, vuelva a indicar a Mathcad la ubicación del fichero de licencia.

Si ha obtenido la licencia de Mathcad de forma automática, puede buscar su fichero de licencia (license.dat) en la carpeta License de su instalación de Mathcad.

Para salir del asistente, pulse Finalizar.
