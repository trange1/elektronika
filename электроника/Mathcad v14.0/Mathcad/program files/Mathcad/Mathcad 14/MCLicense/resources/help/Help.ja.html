Mathcad ライセンスセットアップウィザードでは、インストールした Mathcad が使用可能になるよう、ライセンスを設定します。必要に応じて、ライセンスファイルを請求するプロセスに進みます。

エンタープライズエディションユーザーは、名前と、尋ねられた場合にはライセンスサーバーのポート番号を指定する必要があります。

シングルユーザーエディションユーザーは、PTC ライセンス管理サイトに進みます。PTC ユーザーアカウントを使用してログオンし、ライセンスファイルを請求してください。PTC ユーザーアカウントがない場合、ライセンスファイルを請求するときに作成できます。ウィザードによって、ライセンスファイルが自動的に保存され、Mathcad の起動時にはこのライセンスファイルを検索するよう設定されます。有効なライセンスファイルがすでにある場合、Mathcad がそのライセンスファイルを検索するようこのウィザードで設定できます。

ライセンスファイルまたはライセンスサーバーの場所を一度設定したら、Mathcad が起動するたびにこの情報が使用されます。Mathcad がライセンスファイルまたはライセンスサーバーを見つけられない場合、このウィザードが起動します。その場合、ライセンスファイルの場所を再び設定します。

セットアップメカニズム

このページはシングルユーザーエディションのユーザーのみを対象としています。エンタープライズエディションのユーザーにこのページが表示された場合、システム管理者にお問い合わせください。

PTC からライセンスファイルを入手する必要があるか、または既存のライセンスファイルを使用するよう Mathcad を設定するかを指定します。

PTC からライセンスファイルを入手するには、ワークステーションからインターネットにアクセス可能であり、PTC ユーザーアカウントを持っている必要があります。PTC ユーザーアカウントがない場合、ライセンスファイルを請求するときに作成できます。

PTC ユーザーアカウントについて

PTC ユーザーアカウントにより、ライセンス管理サイトを含む PTC Web サイト (PTC.com) のサポート機能へのアクセスが許可されます。さらに、イベントや Web キャストへの登録、無料ダウンロード、PTC 製品に関するヒントやコツ、チュートリアルへのアクセスが行えます。

Mathcad 製品コード

このページはシングルユーザーエディションのユーザーのみを対象としています。エンタープライズエディションのユーザーにこのページが表示された場合、システム管理者にお問い合わせください。

Mathcad 購入時に入手した製品コードを指定します。Mathcad の評価版を入手したために製品コードがない場合、「trial license」オプションを選択します。

「trial license」オプションを選択した場合、評価期間が終了するまでは Mathcad が正常に起動するように設定されます。評価期間が終了すると、ライセンスファイルまたはライセンスサーバーが尋ねられます。

評価期間が終了する前に Mathcad のライセンスを入手したか入手しようとする場合、コマンドラインから、次のように /license という文字を付けて Mathcad を起動します。

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

Mathcad ライセンスセットアップウィザードが起動し、インストールした Mathcad が使用可能になるようにライセンスを設定できます。

アカウントの作成

PTC ユーザーアカウントがすでにある場合、「次へ」をクリックして「ユーザー名」と「パスワード」を入力します。ライセンスセットアップウィザードはこの情報を使用して PTC Web サイトにアクセスし、Mathcad のライセンスファイルを自動的に入手します。

PTC ユーザーアカウントがない場合、「Create Account」をクリックするとブラウザウィンドウが開いて PTC Web サイトが表示されるので、ここでアカウントを作成できます。アカウントを作成したら、ライセンスセットアップウィザードに戻り、「次へ」をクリックし、アカウントを使用してライセンスファイルを請求します。

PTC ユーザーアカウントについて

Mathcad のライセンスを入手し、PTC Web サイト (PTC.com) のその他の製品サポート機能にアクセスするためには、PTC ユーザーアカウントが必要です。

アカウント情報

指定された場所に PTC の「ユーザー名」と「パスワード」を入力します。ウィザードはこの情報を使用して PTC Web サイトにアクセスし、Mathcad のライセンスファイルを自動的に請求します。

「完了」をクリックしてライセンスファイルを請求します。Mathcad インストールディレクトリにある licenses フォルダにライセンスファイルが自動的に保存され、Mathcad の起動時にはこのファイルを検索するように設定されます。

PTC ユーザーアカウントがない場合、「戻る」をクリックしてウィザードの「Create Account」ページに戻り、アカウントを作成できます。

ライセンスの場所

シングルユーザーエディションユーザー: ライセンスファイルの場所を入力するか、「...」をクリックしてライセンスファイルをブラウズします。「完了」をクリックして、Mathcad の設定を終了します。

エンタープライズエディションユーザー: Mathcad のライセンスファイルまたはライセンスサーバーを指定します。ほとんどのユーザーは「Use a license server」を選択します。ライセンスサーバーを選択した場合、次の手順を実行します。

  1. ライセンスサーバーの名前を指定します。
  2. ライセンスサービスで使用されるポート番号を指定します。デフォルト値 (7788) が自動的に表示されます。

サーバー上の指定したポートを介して、サーバーとワークステーション間の通信が発生します。正確なポート番号がわからない場合は、システム管理者にお問い合わせください。

システム管理者からワークステーション用のライセンスファイルが提供されている場合、「Use a license file」を選択します。ライセンスファイルの場所を入力するか、「...」をクリックしてライセンスファイルをブラウズします。

ライセンスサーバーまたはライセンスファイルを指定したら、「完了」をクリックして、Mathcad の設定を終了します。

手動によるライセンスファイルの請求

Mathcad を実行するコンピュータからインターネットにアクセスできない場合、ウィザードからではなく、手動でライセンスファイルを請求できます。

ライセンスファイルを手動で請求するには、PTC ライセンス管理部門にご連絡ください。電話番号の一覧が PTC Web サイト (http://www.ptc.com/olm/contacts.htm) に掲載されています。

完了しました !

ライセンスサーバーまたはライセンスファイルの場所が適切に記録されました。次に Mathcad を起動するときにはこの情報が使用されます。Mathcad がライセンスファイルまたはライセンスサーバーを見つけられない場合、このウィザードが起動します。その場合、ライセンスファイルの場所を再び設定します。

Mathcad のライセンスファイルを自動的に入手した場合、ライセンスファイル (license.dat) は Mathcad のインストールディレクトリにある License フォルダに格納されています。

ウィザードを終了するには、「完了」をクリックします。
