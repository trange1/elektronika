Page de bienvenue

L'assistant de configuration de licence Mathcad vous permet d'installer la licence votre logiciel Mathcad. Le cas échéant, il vous guidera à travers le processus de demande du fichier de licence.

Les utilisateurs de l'édition Enterprise doivent indiquer le nom - et le cas échéant, le numéro de port - de leur serveur de licences lorsqu'ils y sont invités.

Les utilisateurs de l'édition mono-utilisateur sont dirigés vers le site de gestion des licences PTC. Utilisez votre compte utilisateur PTC pour vous connecter et demander votre fichier de licence. Si vous ne disposez pas de compte utilisateur PTC, vous pouvez en créer un lorsque vous demandez votre fichier de licence. L'assistant enregistre automatiquement votre fichier de licence et configure Mathcad afin qu'il le recherche au démarrage. Si vous disposez déjà d'un fichier de licence valide, vous pouvez utiliser cet assistant afin que Mathcad pointe sur lui.

Une fois que Mathcad connaît l'emplacement du fichier de licence ou du serveur, il utilise cette information à chaque fois qu'il démarre. Si Mathcad ne trouve pas votre serveur ou votre fichier de licence, il lance cet Assistant. Si cela se reproduit, pointez Mathcad vers le fichier de licence.

Mécanisme de configuration

Cette page est réservée aux utilisateurs de l'édition mono-utilisateur. Les utilisateurs de l'édition Enterprise qui arrivent sur cette page doivent contacter leur administrateur système.

Indiquez si vous nécessitez l'obtention un fichier de licence auprès de PTC ou configurez Mathcad afin qu'il utilise un fichier de licence existant.

Pour obtenir un fichier de licence auprès de PTC, vous devez disposer d'un accès à Internet sur la station de travail et d'un compte utilisateur PTC. Si vous ne disposez pas de compte utilisateur PTC, vous pouvez en créer un lorsque vous demandez votre fichier de licence.

A propos des comptes utilisateurs PTC

Votre compte utilisateur PTC vous accorde le droit d'accéder aux fonctionnalités d'assistance du site Web de PTC (PTC.com), notamment à la gestion des licences. De plus, vous pouvez vous enregistrer pour les manifestations et les télédiffusions, recevoir des téléchargements gratuits et découvrir des astuces, des conseils et des didacticiels concernant les produits PTC.

Code produit Mathcad

Cette page est réservée aux utilisateurs de l'édition mono-utilisateur. Les utilisateurs de l'édition Enterprise qui arrivent sur cette page doivent contacter leur administrateur système.

Indiquez le Code produit que vous avez reçu lors de l'achat de Mathcad. Si vous n'avez pas reçu de code produit parce que vous avez reçu une copie d'évaluation de Mathcad, sélectionnez l'option licence d'essai.

Si vous avez sélectionné l'option licence d'essai, l'assistant configure Mathcad afin qu'il se lance normalement jusqu'à la fin de la période d'essai. Une fois cette période terminée, Mathcad vous demandera un fichier ou un serveur de licences.

Si vous obtenez ou souhaitez obtenir une licence pour Mathcad avant la fin de la période d'essai, lancez Mathcad depuis une ligne de commande, suivi du texte " /license" :

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

L'assistant de configuration de licence Mathcad démarre et vous permet d'installer votre licence du logiciel Mathcad.

Création d'un compte

Si vous possédez déjà un compte utilisateur PTC, cliquez sur Suivant pour saisir votre Nom d'utilisateur et votre Mot de passe. L'assistant de configuration de licence utilise ces informations pour accéder au site Web de PTC et obtenir automatiquement un fichier de licence pour Mathcad.

Si vous ne possédez pas de compte utilisateur PTC, cliquez sur Création d'un compte pour ouvrir une fenêtre du navigateur pointant sur le site Web de PTC, où vous pourrez créer un compte. Une fois votre compte créé, retournez dans l'assistant de configuration de licence et cliquez sur Suivant pour demander un fichier de licence à l'aide de votre compte.

A propos des comptes utilisateurs PTC

Le compte utilisateur PTC est nécessaire pour obtenir une licence pour Mathcad. Il permet également d'accéder aux autres fonctionnalités d'assistance du produit disponibles sur le site Web de PTC (PTC.com).

Informations sur le compte

Saisissez votre Nom d'utilisateur PTC et votre Mot de passe dans les emplacements prévus. L'assistant utilise ces informations pour accéder au site Web de PTC et demander automatiquement un fichier de licence pour Mathcad.

Cliquez sur Terminer pour demander votre fichier de licence. L'assistant enregistre automatiquement le fichier de licence dans le dossier des licences de votre installation de Mathcad et configure Mathcad afin qu'il recherche ce fichier au démarrage.

Si vous ne possédez pas de compte utilisateur PTC, cliquez sur Précédent pour revenir à la page Création d'un compte de l'assistant où vous pourrez créer votre compte.

Emplacement du fichier de licence

Edition mono-utilisateur : Entrez l'emplacement de votre fichier de licence ou cliquez sur ... pour naviguer jusqu'à ce fichier. Cliquez sur Terminer pour configurer Mathcad.

Utilisateurs de l'édition Enterprise : Désignez un fichier de licence ou un serveur de licences pour Mathcad. La plupart des utilisateurs sélectionnent Utiliser un serveur de licences. Si vous sélectionnez un serveur de licences :

  1. Indiquez le nom du serveur de licences.
  2. Indiquez le numéro du port utilisé par le service des licences. La valeur par défaut (7788) apparaît automatiquement.

La communication entre le serveur et la station de travail s'établit via le port indiqué du serveur. Si vous ne connaissez pas le numéro de port correct, contactez votre administrateur système.

Si votre administrateur système vous a fourni un fichier de licence pour votre station de travail, sélectionnez Utiliser un fichier de licence. Entrez l'emplacement de votre fichier de licence ou cliquez sur ... pour naviguer jusqu'à ce fichier.

Une fois que vous avez précisé le serveur ou le fichier de licences, cliquez sur Terminer pour configurer Mathcad.

Instructions d'obtention manuelle d'une licence

Si vous ne disposez pas d'un accès à Internet sur l'ordinateur qui exécute Mathcad, vous pouvez demander un fichier de licence manuellement au lieu d'utiliser l'assistant.

Pour ce faire, contactez le service de gestion des licences PTC. La liste des numéros de téléphone est disponible sur le site Web de PTC Web à l'adresse http://www.ptc.com/olm/contacts.htm.

Vous avez terminé !

Le serveur de licences ou le fichier a été enregistré correctement. Mathcad utilise ces informations à chaque fois que vous le lancez. Si Mathcad ne trouve pas votre serveur ou votre fichier de licence, il lance cet Assistant. Si cela se reproduit, pointez Mathcad vers le fichier de licence.

Si vous avez obtenu votre licence Mathcad automatiquement, vous trouverez le fichier de licence (license.dat) dans le dossier License de l'installation de Mathcad.

Pour quitter l'assistant, cliquez sur Terminer.
