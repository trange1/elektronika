시작 페이지

Mathcad 라이센스 설치 마법사를 통해 설치된 Mathcad의 라이센스를 획득할 수 있습니다. 또한 필요한 경우 마법사에서 라이센스 파일 요청 프로세스를 단계별로 안내합니다.

엔터프라이즈 버전 사용자의 경우 관련 메시지가 나타나면 라이센스 서버의 이름을 지정하고 필요에 따라 포트 번호도 지정해야 합니다.

단일 사용자 버전 사용자의 경우 PTC 고객 지원 사이트로 연결됩니다. 이 경우 PTC 사용자 계정을 사용하여 로그인한 후 라이센스 파일을 요청하십시오. PTC 사용자 계정이 없으면 라이센스 파일을 요청할 때 새로 만들 수 있습니다. 마법사에서는 라이센스 파일을 자동으로 저장하고 Mathcad가 시작될 때 이 파일을 찾도록 Mathcad를 구성합니다. 유효한 라이센스 파일이 있으면 이 마법사에서 해당 파일을 지정할 수 있습니다.

라이센스 파일 또는 서버의 위치를 지정한 후에는 Mathcad를 실행할 때마다 이 정보가 사용됩니다. Mathcad에서 라이센스 파일 또는 서버를 찾을 수 없으면 이 마법사가 실행됩니다. 이 경우 라이센스 파일 위치를 다시 지정해야 합니다.

설치 메커니즘

이 페이지는 단일 사용자 버전 사용자에게만 해당됩니다. 엔터프라이즈 버전 사용자에게 이 페이지가 표시되는 경우에는 시스템 관리자에게 문의하십시오.

PTC로부터 라이센스 파일을 획득할지 아니면 기존 라이센스 파일을 사용하도록 Mathcad를 구성할지를 지정합니다.

PTC에서 라이센스 파일을 획득하려면 워크스테이션이 인터넷에 연결되어 있어야 하고 PTC 사용자 계정이 필요합니다. PTC 사용자 계정이 없으면 라이센스 파일을 요청할 때 새로 만들 수 있습니다.

PTC 사용자 계정 정보

PTC 사용자 계정이 있으면 라이센스 관리 기능을 비롯하여 PTC 웹 사이트(PTC.com)의 지원 기능을 사용할 수 있습니다. 또한 이벤트와 웹캐스트에 등록하고 무료 다운로드를 제공 받고 PTC 제품의 팁, 비결, 자습서 등을 검색할 수 있습니다.

Mathcad 제품 코드

이 페이지는 단일 사용자 버전 사용자에게만 해당됩니다. 엔터프라이즈 버전 사용자에게 이 페이지가 표시되는 경우에는 시스템 관리자에게 문의하십시오.

Mathcad 구입 시에 제공된 제품 코드를 입력합니다. 제품 코드가 제공되지 않는 Mathcad 평가판을 사용하려면 평가판 라이센스(trial license) 옵션을 선택합니다.

평가판 라이센스(trial license) 옵션을 선택하면 평가 기간이 끝날 때까지 정상적으로 실행되도록 Mathcad가 구성됩니다. 그리고 평가 기간이 끝나면 라이센스 파일이나 서버를 묻는 메시지가 나타납니다.

평가 기간이 끝나기 전에 Mathcad 라이센스를 획득하려면 명령줄에서 다음과 같이 뒤에 " /license"라는 텍스트를 사용하여 Mathcad를 실행합니다.

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

이렇게 하면 Mathcad 라이센스 설치 마법사가 실행되며, 마법사의 안내에 따라 설치된 Mathcad에 대한 라이센스를 지정할 수 있습니다.

계정 만들기

PTC 사용자 계정이 있으면 Next를 클릭하여 User Name과 Password를 입력합니다. 라이센스 설치 마법사에서 이 정보를 사용하여 PTC 웹 사이트에 액세스하고 Mathcad 라이센스 파일을 자동으로 가져옵니다.

PTC 사용자 계정이 없으면 Create Account를 클릭하여 브라우저 창을 열고 PTC 웹 사이트로 이동한 다음 계정을 만듭니다. 계정을 만들었으면 라이센스 설치 마법사로 돌아와 Next를 클릭하고 새로 만든 계정을 사용하여 라이센스 파일을 요청합니다.

PTC 사용자 계정 정보

Mathcad 라이센스를 가져오고 PTC 웹 사이트(PTC.com)의 다른 제품 지원 기능을 사용하려면 PTC 사용자 계정이 필요합니다.

계정 정보

입력란에 PTC User Name과 Password를 입력합니다. 마법사에서 이 정보를 사용하여 PTC 웹 사이트에 액세스하고 Mathcad 라이센스 파일을 자동으로 요청합니다.

Finish를 클릭하여 라이센스 파일을 요청합니다. Mathcad 설치 폴더의 licenses 폴더에 라이센스 파일이 자동으로 저장되고 시작할 때 이 파일을 찾도록 Mathcad가 구성됩니다.

PTC 사용자 계정이 없으면 Back을 클릭하여 Create Account 페이지로 돌아가 계정을 만듭니다.

라이센스 위치

단일 사용자 버전 사용자 - 라이센스 파일 위치를 입력하거나 ...를 클릭하여 해당 파일을 탐색합니다. Finish를 클릭하여 Mathcad를 구성합니다.

엔터프라이즈 버전 사용자 - Mathcad에 사용할 라이센스 파일 또는 라이센스 서버를 지정합니다. 대부분의 사용자는 Use a license server를 선택하면 됩니다. 라이센스 서버를 선택한 경우 다음과 같이 합니다.

  1. 라이센스 서버의 이름을 지정합니다.
  2. 라이센스 서비스에 사용되는 포트 번호를 지정합니다. 이때 기본값(7788)이 자동으로 표시됩니다.

서버에서 지정된 포트를 통해 서버와 워크스테이션 간의 통신이 이루어집니다. 올바른 포트 번호를 모르는 경우 시스템 관리자에게 문의하십시오.

시스템 관리자가 워크스테이션에서 사용할 라이센스 파일을 제공한 경우 Use a license file을 선택합니다. 라이센스 파일 위치를 입력하거나 ...를 클릭하여 해당 파일을 찾습니다.

라이센스 서버 또는 파일을 지정했으면 Finish를 클릭하여 Mathcad를 구성합니다.

수동으로 라이센스를 얻을 경우의 지침

Mathcad를 실행하는 컴퓨터가 인터넷에 연결되어 있지 않은 경우에는 마법사를 통하지 않고 수동으로 라이센스 파일을 요청할 수 있습니다.

수동으로 라이센스 파일을 요청하려면 PTC 고객 지원 부서에 문의하십시오. 관련 전화 번호 목록은 PTC 웹 사이트 http://www.ptc.com/olm/contacts.htm에 나와 있습니다.

라이센스 관련 작업 완료

라이센스 서버 또는 파일 위치가 정상적으로 기록되었습니다. 다음에 Mathcad를 실행하면 이 정보가 사용됩니다. Mathcad에서 라이센스 파일 또는 서버를 찾을 수 없으면 이 마법사가 실행됩니다. 이 경우 라이센스 파일 위치를 다시 지정해야 합니다.

자동으로 라이센스를 얻고 나면 Mathcad 설치 폴더의 License 폴더에 라이센스 파일(license.dat)이 만들어집니다.

Finish를 클릭하여 마법사를 종료합니다.
