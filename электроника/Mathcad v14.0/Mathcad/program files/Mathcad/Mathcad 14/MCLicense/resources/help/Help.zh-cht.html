歡迎使用頁面

「Mathcad 許可證設定精靈」可讓您授權 Mathcad 安裝。必要時，還可以引導您取得許可證檔案。

企業版本使用者應該在出現提示時，指定許可證伺服器的名稱，必要時，再加上連接埠號碼。

單一使用者版本的使用者會被引導到「PTC 許可證管理網站」。請使用 PTC 使用者帳戶登入，要求許可證檔案。若沒有 PTC 使用者帳戶，您可以在要求許可證檔案時建立帳戶。精靈會自動儲存您的許可證檔案，並且配置 Mathcad 在啟動時尋找此檔。若已經有有效的許可證檔案，您可以使用此精靈，為 Mathcad 指出該檔案。

Mathcad 知道許可證檔案或伺服器的位置後，便會在每次啟動時使用此資訊。若 Mathcad 找不到許可證檔案或伺服器，則會啟動此精靈。此時，請再度為 Mathcad 指出許可證檔案。

設定機制

這個頁面只適用於單一使用者版本的使用者。企業版本的使用者看見此頁面時，請連絡系統管理員。

請指定您要向 PTC 取得許可證檔案，或是配置 Mathcad 使用現有的許可證檔案。

欲向 PTC 索取許可證檔案，您必須具備能夠存取網際網路的工作站以及 PTC 使用者帳戶。若沒有 PTC 使用者帳戶，您可以在要求許可證檔案時建立帳戶。

關於 PTC 使用者帳戶

PTC 使用者帳戶可讓您使用 PTC 網站 (PTC.com) 的支援功能，包括許可證管理。此外，您可以註冊活動和網路廣播、獲得免費下載、探索 PTC 產品的秘訣、技巧和教學專區。

Mathcad 產品代碼

這個頁面只適用於單一使用者版本的使用者。企業版本的使用者看見此頁面時，請連絡系統管理員。

請指定您購買 Mathcad 時取得的產品代碼。若您取得的是沒有產品代碼的 Mathcad 試用版，請選取「試用許可證」選項。

若您選取「試用許可證」選項，精靈會配置 Mathcad 正常啟動，直到試用期滿。試用期滿後，Mathcad 會提示要求許可證檔案或伺服器。

若您在試用期滿前已經取得或是想取得 Mathcad 許可證，請從指令行啟動 Mathcad，並且在後面加上文字 " /license"：

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

「Mathcad 許可證設定精靈」隨即啟動，讓您授權 Mathcad 安裝。

建立帳戶

若您已有 PTC 使用者帳戶，請按「下一步」，輸入「使用者名稱」和「密碼」。「許可證設定精靈」會使用此資訊存取 PTC 網站，自動取得 Mathcad 許可證檔案。

若您沒有 PTC 使用者帳戶，請按一下「建立帳戶」，開啟一個指向 PTC 網站的瀏覽器視窗，在此建立帳戶。建立帳戶後，回到「許可證設定精靈」，按「下一步」，使用您的帳戶要求許可證檔案。

關於 PTC 使用者帳戶

您需要使用 PTC 使用者帳戶來取得 Mathcad 許可證及存取 PTC 網站 (PTC.com) 的其他產品支援功能。

帳戶資訊

在提供的空格中，輸入「使用者名稱」和「密碼」。精靈會使用此資訊存取 PTC 網站，自動要求 Mathcad 許可證檔案。

請按一下「完成」，取得許可證檔案。精靈會自動將許可證檔案儲存到 Mathcad 安裝的許可證資料夾，並且配置 Mathcad 在啟動時尋找這個檔案。

若您沒有 PTC 使用者帳戶，請按「上一步」，回到精靈的「建立帳戶」頁面，在此建立帳戶。

許可證位置

單一使用者版本的使用者：輸入許可證檔案的位置，或按一下「...」，瀏覽至檔案。按一下「完成」，配置 Mathcad。

企業版本的使用者：指定 Mathcad 的許可證檔案或許可證伺服器。大部分的使用者應選取「使用許可證伺服器」。若您選取許可證伺服器：

  1. 指定許可證伺服器的名稱。
  2. 指定許可證服務使用的連接埠號碼。隨即自動出現預設值 (7788)。

伺服器和工作站會在伺服器的指定連接埠上通訊。若您不知道正確的連接埠號碼，請連絡系統管理員。

若系統管理員提供工作站的許可證檔案，請選取「使用許可證檔案」。輸入許可證檔案的位置，或按一下「...」，瀏覽至檔案。

指定許可證伺服器或許可證檔案後，按一下「完成」，配置 Mathcad。

手動授權指示

若您在執行 Mathcad 的電腦上沒有網際網路存取權，您可以手動要求許可證檔案，不必透過精靈。

欲手動要求許可證檔案，請連絡「PTC 許可證管理」。PTC 網站 http://www.ptc.com/olm/contacts.htm 提供可用的電話號碼清單。

完成！

許可證伺服器或檔案位置已記錄妥當。Mathcad 會在下次啟動時使用此資訊。若 Mathcad 找不到許可證檔案或伺服器，則會啟動此精靈。此時，請再度為 Mathcad 指出許可證檔案。

若自動授權 Mathcad，您現在可以在 Mathcad 安裝的「許可證」資料夾中，找到許可證檔案 (license.dat)。

欲結束精靈，請按一下「完成」。
