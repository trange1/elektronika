欢迎页

“Mathcad 许可证安装”向导允许授权安装 Mathcad。如有必要，它可指导用户进入申请许可证文件的过程。

当出现提示时，企业版用户应指定其许可证服务器的名称和端口号（如有必要）。

单用户版本的用户可被指导进入 PTC 许可证管理站点。请使用 PTC 用户帐户登录和申请许可证文件。如果没有 PTC 用户帐户，可在申请许可证文件时创建一个用户帐户。向导将自动保存许可证文件和配置 Mathcad，以便在启动时搜索该文件。如果已拥有有效的许可证文件，则可使用该向导将 Mathcad 指向该文件。

Mathcad 知道许可证文件或服务器的位置后，它将在每次启动时使用该信息。如果 Mathcad 无法找到许可证文件或服务器，则将启动该向导。如果发生这种情况，请再次将 Mathcad 指向该许可证文件。

安装机制

本页仅适用于单用户版本的用户。访问该页面的企业版用户应联系其系统管理员。

指定是否需要从 PTC 获取许可证文件或配置 Mathcad 以使用现有许可证文件。

要从 PTC 获取许可证文件，用户必须能够通过工作站访问 Internet 以及一个 PTC 用户帐户。如果没有 PTC 用户帐户，可在申请许可证文件时创建一个用户帐户。

关于 PTC 用户帐户

PTC 用户帐户允许用户访问 PTC Web 站点 (PTC.com) 的支持功能，包括许可证管理。此外，用户还可在该站点上注册事件和 Webcast，接收免费下载，以及获得有关 PTC 产品的提示、使用技巧和教程。

Mathcad 产品代码

本页仅适用于单用户版本的用户。访问该页面的企业版用户应联系其系统管理员。

购买 Mathcad 时，请指定已接收的产品代码。如果用户因为接收了 Mathcad 的评估版本而导致未接收到产品代码，请选择“试用版许可证”选项。

如果选择了“试用版许可证”选项，向导则会配置 Mathcad 以正常启动，直到评估期结束。评估期结束后，Mathcad 将提示用户使用许可证文件或服务器。

如果在评估期结束前已获取或打算获取 Mathcad 许可证，则可通过其后紧跟文本 " /license" 的命令行来启动 Mathcad：

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

“Mathcad 许可证安装”向导启动并允许授权安装 Mathcad。

创建帐户

如果已拥有 PTC 用户帐户，请单击“下一步”以输入“用户名”和“密码”。“许可证安装”向导将使用该信息访问 PTC Web 站点，并自动获取 Mathcad 许可证文件。

如果没有 PTC 用户帐户，请单击“创建帐户”以打开指向 PTC Web 站点的浏览器窗口，用户可在该窗口中创建帐户。创建帐户后，请使用帐户返回“许可证安装”向导，然后单击“下一步”以申请许可证文件。

关于 PTC 用户帐户

要获取 Mathcad 许可证和访问 PTC Web 站点 (PTC.com) 的其他产品支持功能，必需拥有 PTC 用户帐户。

帐户信息

在提供的空格中输入 PTC“用户名”和“密码”。向导将使用该信息访问 PTC Web 站点，并自动申请 Mathcad 许可证文件。

单击“完成”以申请许可证文件。向导会自动将许可证文件保存到 Mathcad 安装的许可证文件夹中，并配置 Mathcad 以便在启动时搜索该文件。

如果没有 PTC 用户帐户，请单击“返回”以返回到向导的“创建帐户”页面，用户可在该页面中创建帐户。

许可证位置

单用户版本的用户：输入许可证文件的位置或单击 "..." 以浏览至许可证文件。单击“完成”以配置 Mathcad。

企业版用户：指定 Mathcad 许可证文件或许可证服务器。大多数用户选择“使用许可证服务器”。如果选择许可证服务器：

  1. 指定许可证服务器的名称。
  2. 指定许可证服务使用的端口号。自动出现默认值 (7788)。

服务器和工作站通过服务器上的指定端口进行通信。如果用户不知道正确的端口号，请联系系统管理员。

如果系统管理员提供了工作站的许可证文件，请选择“使用许可证文件”。输入许可证文件的位置或单击 "..." 以浏览至许可证文件。

指定许可证服务器或文件后，单击“完成”以配置 Mathcad。

手动授权说明

如果运行 Mathcad 的计算机无法访问 Internet，用户则可手动申请许可证文件，而不是通过向导进行申请。

要手动申请许可证文件，请联系 PTC 许可证管理。PTC Web 站点 http://www.ptc.com/olm/contacts.htm 上列出了可用的电话号码。

操作已完成！

已正确记录许可证服务器或文件位置。Mathcad 将在下次启动时使用该信息。如果 Mathcad 无法找到许可证文件或服务器，则将启动该向导。如果发生这种情况，请再次将 Mathcad 指向该许可证文件。

如果已自动授权 Mathcad，则用户现在便可在 Mathcad 安装的“许可证”文件夹中找到许可证文件 (license.dat)。

要退出向导，请单击“完成”。
