Welcome Page

The Mathcad License Setup wizard allows you to license your installation of Mathcad. If necessary, it directs you through the process of requesting your license file.

Enterprise-edition users should specify the name - and, if necessary, port number - of their license server when prompted.

Single-user-edition users are directed to the PTC License Management site. Use your PTC user account to log on and request your license file. If you do not have a PTC user account, you can create one when you request your license file. The wizard automatically saves your license file and configures Mathcad to look for it at startup. If you already have a valid license file, you can use this wizard to point Mathcad to it.

Once Mathcad knows the location of your license file or server, it uses this information every time it launches. If Mathcad cannot find your license file or server, it launches this wizard. Should this happen, again point Mathcad to the license file.

Setup Mechanism

This page is for single-user-edition users only. Enterprise-edition users who reach this page should contact their system administrator.

Specify whether you need to obtain a license file from PTC or configure Mathcad to use an existing license file.

To obtain a license file from PTC, you must have Internet access on the workstation and a PTC user account. If you do not have a PTC user account, you can create one when you request your license file.

About PTC user accounts

Your PTC user account grants you access to the support features of the PTC Web site (PTC.com) including license management. In addition, you can register for events and Webcasts, receive free downloads, and discover tips, tricks, and tutorials for PTC products.

Mathcad Product Code

This page is for single-user-edition users only. Enterprise-edition users who reach this page should contact their system administrator.

Specify the Product Code you received when you purchased Mathcad. If you did not receive a product code because you received an evaluation copy of Mathcad, select the trial license option.

If you selected the trial license option, the wizard configures Mathcad to launch normally until the end of the evaluation period. Once the evaluation period ends, Mathcad will prompt you for a license file or server.

If you obtain, or intend to obtain, a license for Mathcad before the evaluation period ends, launch Mathcad from a command line, followed by the text " /license":

"C:\Program Files\Mathcad\Mathcad 14\mathcad.exe" /license

The Mathcad License Setup wizard launches and allows you to license your installation of Mathcad.

Create Account

If you already have a PTC user account, click Next to enter your User Name and Password. The License Setup wizard uses this information to access the PTC Web site and automatically obtain a license file for Mathcad.

If you do not have a PTC user account, click Create Account to open a browser window pointing to the PTC Web site, where you can create an account. Once you create your account, return to the License Setup wizard and click Next to request a license file using your account.

About PTC user accounts

A PTC user account is required to obtain a license for Mathcad and to access other product support features of the PTC Web site (PTC.com).

Account Information

Enter your PTC User Name and Password in the spaces provided. The wizard uses this information to access the PTC Web site and automatically request a license file for Mathcad.

Click Finish to request your license file. The wizard automatically saves the license file to the licenses folder of your Mathcad installation and configures Mathcad to look for this file at startup.

If you do not have a PTC user account, click Back to return to the Create Account page of the wizard, where you can create your account.

License Location

Single-user-edition users: Enter the location of your license file or click ... to browse to it. Click Finish to configure Mathcad.

Enterprise-edition users: Specify either a license file or a license server for Mathcad. Most users select Use a license server. If you select a license server:

  1. Specify the name of the license server.
  2. Specify the port number used by the license service. The default value (7788) appears automatically.

Communication between the server and workstation occurs over the specified port on the server. If you do not know the correct port number, contact your system administrator.

If your system administrator has provided you with a license file for your workstation, select Use a license file. Enter the location of your license file or click ... to browse to it.

Once you specify the license server or file, click Finish to configure Mathcad.

Manual Licensing Instructions

If you do not have Internet access on the computer that runs Mathcad, you can request a license file manually, rather than through the wizard.

To manually request a license file, contact PTC License Management. A listing of phone numbers is available on the PTC Web site at http://www.ptc.com/olm/contacts.htm.

You Are Done!

The license server or file location has been recorded properly. Mathcad uses this information the next time you launch it. If Mathcad cannot find your license file or server, it launches this wizard. Should this happen, again point Mathcad to the license file.

If you licensed Mathcad automatically, you can now find your license file (license.dat) in the License folder of your Mathcad installation.

To exit the wizard, click Finish.
