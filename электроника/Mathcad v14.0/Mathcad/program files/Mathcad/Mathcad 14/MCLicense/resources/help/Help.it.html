Pagina iniziale

La procedura guidata Mathcad License Setup (Installazione della licenza Mathcad) consente di ottenere una licenza per l'installazione di Mathcad. Se necessario, guida l'utente nel processo di richiesta del file di licenza.

Alla richiesta, gli utenti dell'edizione Enterprise devono specificare il nome e, se necessario, il numero di porta del server licenze.

Gli utenti dell'edizione per singolo utente vengono indirizzati al sito di gestione licenze di PTC. Utilizzare l'account utente PTC per accedere e richiedere il file di licenza. Se non si dispone di un account utente PTC, è possibile crearne uno al momento di richiedere il file di licenza. La procedura guidata salva automaticamente il file di licenza e configura Mathcad per ricercarlo all'avvio. Se si dispone già di un file di licenza valido, è possibile utilizzare la procedura guidata per far sì che Mathcad possa individuarlo.

Una volta conosciuta la posizione del file di licenza o del server, Mathcad utilizza queste informazioni a ogni avvio. Se non trova il file di licenza o il server, viene avviata la procedura guidata. In tal caso, seguire la procedura affinché Mathcad possa individuarlo.

Meccanismo di impostazione

Questa pagina si riferisce alla sola edizione per singolo utente. Gli utenti dell'edizione Enterprise che raggiungono questa pagina devono contattare l'amministratore di sistema.

Specificare se è necessario ottenere un file di licenza da PTC o configurare Mathcad per l'uso di un file di licenza esistente.

Per ottenere un file di licenza da PTC, è necessario avere accesso a Internet sulla workstation e disporre di un account utente PTC. Se non si dispone di un account utente PTC, è possibile crearne uno al momento di richiedere il file di licenza.

Informazioni sugli account utente PTC

L'account utente PTC consente di accedere alle funzioni di supporto del sito Web PTC (PTC.com), compresa la gestione delle licenze. Inoltre, consente di registrarsi per eventi e Webcast, ricevere download gratuiti e scoprire suggerimenti, trucchi ed esercitazioni per i prodotti PTC.

Codice prodotto di Mathcad

Questa pagina si riferisce alla sola edizione per singolo utente. Gli utenti dell'edizione Enterprise che raggiungono questa pagina devono contattare l'amministratore di sistema.

Specificare il Codice prodotto ricevuto all'acquisto di Mathcad. Se non è stato ricevuto un codice prodotto perché la copia di Mathcad è a scopo di valutazione, selezionare l'opzione trial license (licenza di valutazione).

Se è stata selezionata l'opzione trial license, la procedura guidata configura Mathcad per l'avvio normale fino al termine del periodo di valutazione. Successivamente, viene richiesto un file o un server di licenze.

Se si ottiene o si intende ottenere una licenza per Mathcad prima dello scadere del periodo di valutazione, avviare Mathcad da una riga di comando, seguito dal testo " /license":

"C:\Programmi\Mathcad\Mathcad 14\mathcad.exe" /license

Si avvia la procedura guidata Mathcad License Setup, che consente di ottenere una licenza per l'installazione di Mathcad.

Creazione di un account

Se si dispone già di un account utente PTC, fare clic su Next (Avanti) per immettere il Nome utente e la Password. La procedura guidata utilizza queste informazioni per accedere al sito Web PTC e ottenere automaticamente un file di licenza per Mathcad.

Se non si dispone di un account utente PTC, fare clic su Create Account (Crea account) per aprire una finestra del browser che punta al sito Web PTC, dove è possibile creare un account. Al termine della creazione, tornare alla procedura guidata License Setup e fare clic su Next (Avanti) per richiedere un file di licenza utilizzando il proprio account.

Informazioni sugli account utente PTC

L'account utente PTC è necessario per ottenere una licenza per Mathcad e per accedere ad altre funzioni di supporto dei prodotti del sito Web PTC (PTC.com).

Informazioni sugli account

Immettere il Nome utente e la Password PTC negli appositi spazi. La procedura guidata utilizza queste informazioni per accedere al sito Web PTC e richiedere automaticamente un file di licenza per Mathcad.

Fare clic su Finish (Fine) per richiedere il file di licenza. La procedura guidata salva automaticamente il file di licenza nella cartella delle licenze dell'installazione di Mathcad e configura Mathcad per ricercarlo all'avvio.

Se non si dispone di un account utente PTC, fare clic su Back (Indietro) per tornare alla pagina Create Account (Crea account) della procedura guidata, che consente di creare un account.

Posizione della licenza

Edizione per singolo utente: immettere la posizione del file di licenza o fare clic su ... per ricercarlo. Fare clic su Finish (Fine) per configurare Mathcad.

Edizione Enterprise: specificare un file di licenza o un server di licenze per Mathcad. Per la maggior parte degli utenti è necessario selezionare Use a license server (Utilizza un server di licenze). Se si seleziona un server di licenze:

  1. Specificare il nome del server di licenze.
  2. Specificare il numero di porta utilizzato dal servizio licenze. Il valore di default (7788) viene visualizzato automaticamente.

La comunicazione tra il server e la workstation avviene sulla porta specificata del server. Se non si conosce il numero di porta corretto, contattare l'amministratore di sistema.

Se l'amministratore di sistema ha fornito un file di licenza per la workstation, selezionare Use a license file (Utilizza un file di licenza). Immettere la posizione del file di licenza o fare clic su ... per ricercarlo.

Dopo aver specificato il server o il file di licenza, fare clic su Finish (Fine) per configurare Mathcad.

Istruzioni per la configurazione manuale della licenza

Se il computer su cui viene eseguito Mathcad non è connesso a Internet è possibile richiedere manualmente un file di licenza anziché utilizzare la procedura guidata.

A tale scopo, contattare il servizio di gestione delle licenze di PTC. Per un elenco dei numeri di telefono, vedere il sito Web PTC all'indirizzo http://www.ptc.com/olm/contacts.htm.

Il processo è completato

Il server di licenze o la posizione del file sono stati correttamente registrati. Mathcad utilizza queste informazioni al successivo avvio. Se non trova il file di licenza o il server, viene avviata la procedura guidata. In tal caso, seguire la procedura affinché Mathcad possa individuarlo.

Se la licenza di Mathcad è stata ottenuta automaticamente, il file di licenza (license.dat) si trova nella cartella License dell'installazione di Mathcad.

Per uscire dalla procedura guidata, fare clic su Fine.
