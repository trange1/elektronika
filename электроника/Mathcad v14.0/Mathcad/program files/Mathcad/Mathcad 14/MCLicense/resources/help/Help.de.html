Willkommensseite

Der Setup-Assistent für Mathcad-Lizenzen ermöglicht Ihnen die Lizenzierung Ihrer Installation von Mathcad. Falls erforderlich, führt er Sie durch den Vorgang des Anforderns Ihrer Lizenzdatei.

Benutzer der Enterprise Edition sollten bei Aufforderung den Namen und gegebenenfalls die Portnummer des Lizenzservers angeben.

Benutzer der Einzelbenutzerversion werden zur PTC Website für die Lizenzverwaltung geleitet. Melden Sie sich bei Ihrem PTC Benutzerkonto an, und fordern Sie Ihre Lizenzdatei an. Wenn Sie kein PTC Benutzerkonto haben, können Sie dieses beim Anfordern der Lizenzdatei erstellen. Der Assistent speichert Ihre Lizenzdatei automatisch und konfiguriert Mathcad so, dass es beim Programmstart danach sucht. Wenn Sie bereits über eine gültige Lizenzdatei verfügen, können Sie im Setup-Assistenten darauf verweisen.

Wenn Mathcad den Speicherort Ihrer Lizenzdatei oder des Servers kennt, verwendet es diese Informationen beim Programmstart. Wenn Mathcad Ihre Lizenzdatei oder den Server nicht findet, startet es diesen Assistenten. Geben Sie in diesem Fall erneut den Speicherort der Lizenzdatei an.

Setup-Mechanismus

Diese Seite ist nur für Benutzer der Einzelbenutzerversion gedacht. Benutzer der Enterprise Edition, die auf diese Seite gelangen, sollten sich an ihren Systemadministrator wenden.

Geben Sie an, ob Sie eine Lizenzdatei von PTC benötigen, oder ob Sie Mathcad für die Verwendung einer vorhandenen Lizenzdatei konfigurieren möchten.

Um eine Lizenzdatei von PTC anzufordern, benötigen Sie einen Internetzugang auf Ihrer Workstation und ein PTC Benutzerkonto. Wenn Sie kein PTC Benutzerkonto haben, können Sie dieses beim Anfordern der Lizenzdatei erstellen.

PTC Benutzerkonten

Ihr PTC Benutzerkonto gewährt Ihnen Zugang zu den Support-Funktionen der PTC Website (PTC.com) einschließlich der Lizenzverwaltung. Darüber hinaus können Sie sich für Veranstaltungen und Webcasts registrieren, kostenlose Software herunterladen, und Sie finden Tipps und Tricks sowie Lernprogramme für PTC Produkte.

Mathcad-Produkt-Code

Diese Seite ist nur für Benutzer der Einzelbenutzerversion gedacht. Benutzer der Enterprise Edition, die auf diese Seite gelangen, sollten sich an ihren Systemadministrator wenden.

Geben Sie den Produkt-Code ein, den Sie beim Kauf von Mathcad erhalten haben. Wenn Sie keinen Produkt-Code erhalten haben, da Sie eine Testversion von Mathcad haben, wählen Sie die Option Testlizenz.

Wenn Sie die Option Testlizenz ausgewählt haben, konfiguriert der Assistent Mathcad so, dass das Programm bis zum Ablauf des Testzeitraums normal gestartet wird. Nach Ablauf dieses Testzeitraums werden Sie bei Programmstart nach einer Lizenzdatei oder dem Lizenzserver gefragt.

Wenn Sie vor Ablauf des Testzeitraums eine Lizenz für Mathcad erwerben bzw. erwerben möchten, starten Sie Mathcad über die Befehlszeile, und hängen Sie den Text " /license" an den Startbefehl an:

"C:\Programme\Mathcad\Mathcad 14\mathcad.exe" /license

Der Setup-Assistent für Mathcad-Lizenzen wird gestartet und ermöglicht Ihnen nun die Lizenzierung Ihrer Installation von Mathcad.

Kontoerstellung

Wenn Sie bereits ein PTC Benutzerkonto haben, klicken Sie auf Weiter, um Ihren Benutzernamen und Ihr Kennwort einzugeben. Der Lizenz-Setup-Assistent verwendet diese Informationen, um auf die PTC Website zuzugreifen und automatisch eine Lizenzdatei von Mathcad abzurufen.

Wenn Sie kein PTC Benutzerkonto haben, klicken Sie auf Konto erstellen, um ein Browserfenster mit der PTC Website zu öffnen, auf der Sie ein Konto erstellen können. Sobald Sie das Konto erstellt haben, kehren Sie zum Setup-Assistenten für Lizenzen zurück, und klicken Sie auf Weiter, um über Ihr Benutzerkonto eine Lizenzdatei anzufordern.

PTC Benutzerkonten

Ein PTC Benutzerkonto wird benötigt, um eine Lizenz für Mathcad anzufordern und um Zugriff auf andere Produkt-Support-Funktionen auf der PTC Website (PTC.com) zu erhalten.

Informationen zum Benutzerkonto

Geben Sie Ihren PTC Benutzernamen und Ihr Kennwort in die vorgesehenen Felder ein. Der Assistent verwendet diese Informationen, um auf die PTC Website zuzugreifen und automatisch eine Lizenzdatei von Mathcad anzufordern.

Klicken Sie auf Fertig stellen, um die Lizenzdatei anzufordern. Der Assistent speichert die Lizenzdatei automatisch im Lizenzordner Ihrer Mathcad-Installation und konfiguriert Mathcad so, dass es bei Programmstart prüft, ob diese Datei vorhanden ist.

Wenn Sie kein PTC Benutzerkonto haben, klicken Sie auf Zurück, um zur Seite Konto erstellen des Assistenten zurückzukehren, auf der Sie das Benutzerkonto erstellen können.

Speicherort der Lizenz

Benutzer der Einzelbenutzerversion: Geben Sie den Speicherort Ihrer Lizenzdatei ein, oder klicken Sie auf ..., um danach zu suchen. Klicken Sie auf Fertig stellen, um Mathcad zu konfigurieren.

Benutzer der Enterprise Edition: Geben Sie eine Lizenzdatei oder einen Lizenzserver für Mathcad an. Die meisten Benutzer wählen die Option Lizenzserver verwenden. Wenn Sie einen Lizenzserver auswählen:

  1. Geben Sie den Namen des Lizenzservers an.
  2. Geben Sie die vom Lizenzdienst verwendete Portnummer an. Der Standardwert (7788) wird automatisch angezeigt.

Die Kommunikation zwischen dem Server und der Workstation erfolgt über den angegebenen Port des Servers. Wenn Sie die Portnummer nicht kennen, wenden Sie sich an Ihren Systemadministrator.

Wenn Sie von Ihrem Systemadministrator eine Lizenzdatei für Ihre Workstation erhalten haben, wählen Sie Lizenzdatei verwenden. Geben Sie den Speicherort Ihrer Lizenzdatei ein, oder klicken Sie auf ..., um danach zu suchen.

Nachdem Sie den Lizenzserver bzw. die Datei angegeben haben, klicken Sie auf Fertig stellen, um Mathcad zu konfigurieren.

Anweisungen zur manuellen Lizenzierung

Wenn Sie keinen Internetzugang auf dem Computer haben, auf dem Mathcad ausgeführt wird, können Sie die Lizenzdatei manuell statt über den Assistenten anfordern.

Wenden Sie sich hierzu an die PTC Lizenzverwaltung. Die entsprechenden Telefonnummern finden Sie auf der PTC Website unter http://www.ptc.com/olm/contacts.htm.

Fertig!

Der Lizenzserver bzw. der Speicherort der Lizenzdatei wurden ordnungsgemäß registriert. Mathcad verwendet diese Informationen beim nächsten Programmstart. Wenn Mathcad Ihre Lizenzdatei oder den Server nicht findet, startet es diesen Assistenten. Geben Sie in diesem Fall erneut den Speicherort der Lizenzdatei an.

Wenn Sie Mathcad automatisch lizenziert haben, befindet sich die Lizenzdatei (license.dat) im Ordner License Ihrer Mathcad-Installation.

Klicken Sie zum Beenden des Assistenten auf Fertig stellen.
